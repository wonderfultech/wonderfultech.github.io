解法一 滑动窗口+哈希表 p 的字母异位词，只要关心p中各个字母的个数，无需关注字母间的顺序。 在s串上形成一个长度为p.size的窗口，并移动这个窗口，不断计算窗口中各个字符的个数。 当窗口中各个字符的个数与p中各个字符的个数对应相等时，说明窗口形成了一个p的字母异位词。 代码： Hits: 29

leetcode442 数组中重复的数据 Find All Duplicates in an Array

Posted on 2019年11月27日2019年12月4日by jasonpeng

解法一 交换法 关键是用好条件： 1 ≤ a[i] ≤ n （n为数组长度） 。 a[1]-1在[0,n-1]中。将a[i]放在位置a[i]-1是可行的。 遍历元素时， 当 a[i] != i+1的，就将其交换到正确的位置a[i]-1。 此时，当 a[i] == a[a[i]-1]时，表示a[i]有重复。 遍历完后，再遍历一次。所有 a[i] != i+1 的a[i]就是结果。 代码： Hits: 6
