leetcode269 火星词典 Alien Dictionary

Posted on 2019年12月16日2019年12月16日by jasonpeng

解法一 拓扑排序 从已经有序的字典中，推断出字符顺序。 关键是明确字符串比较方法。 字符串A和B比较大小。 在相同位置i处， 当A[i] < B[i]时，A小于B。 因此，反过来，A和B在相同位置i上，只有A[i] != B[i]时，才能有字符串A和B的顺序推断出字符A[i]和B[i]的大小关系。 思路：对字典中的每个字符串A，与其之后的每个字符串B，组合。找到A和B中，第一对，在相同位置i上，A[i] ！= B[i]的两个字符串。确定A[i] < B[i]。并以此构造一个有向边A[i]->B[i]。 由这些有向边，构造了一个有向图G。 此外，不在任何有向边中，但是在字典中的其它字符。它们的顺序并不重要，也可以将它们加入图G。 最后对G拓扑排序。就能明确字符顺序。 构造图 字典中的字符都是小写字母，最多256个字符。 用邻接矩阵就能表示有向图。 拓扑排序算法 用递归方法求拓扑排序。同时，排除环的情况。 按有向边，遍历图中节点u的全部邻接点v。 生成结果 对图中节点，进行拓扑排序。 代码： Hits: 32

leetcode488 祖玛游戏 Zuma Game

Posted on 2019年12月15日2019年12月15日by jasonpeng

解法一 回溯法 思路：取手中的每个字符a，插入桌上的不同位置p。然后尽可能消掉连续三个或三个以上的字符，直到不能再消除为止。然后，递归再求解。 本题的难点在于减枝。 有几种减枝策略： 1，已用的字符数step，已经求得最优球数ans。 当step+1 >= ans时，这次递归即使做了，step+1也不会比ans更少。 2，手里没有字符了。不用再做了。 3，只用手中哪些在桌上已经有得字符。 例如：”WRRBBW”, “RBY” 手中有RBY，三种字符。但是字符Y不在桌子上。字符Y即使插入到桌子中，就不会消掉其它字符，反而多用了一个字符。 4，将手中字符，插入到桌子上相同字符子串得最后一个之后。避免重复插入。 例如：”WRRBBW”, “RBY” 手中字符R，插入到”RR”之后，”BB”之前。在”RR”之前和中间再插入一个R，效果是一样的。 而将R插入到”BB”中间和之后，没有任何用处。 5，插入一次字符，可能会引起多次移除。每次插入后，要多次移除，直到不能再移除为止。 例如：”WBRRBBWW”,”RYB” 当插入R后， “WBRRBBWW” -> “WBRRRBBWW” -> “WBBBWW” -> “WWW” ->”” 代码： Hits: 9
