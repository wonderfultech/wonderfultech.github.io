leetcode43 字符串相乘 Multiply Strings

Posted on 2019年11月23日2019年12月4日by jasonpeng

解法一 竖式乘法 数字A与数字B相乘，A*B的总位数不会超过A和B的位数之和。 A[i] * B[j] 会贡献结果位置[i+j]和[i+j+1]上的值。 例如： num1 = “123”, num2 = “456” A[2] * B[2]= 3 * 6 =18, 18 在结果中的位置是4和5. A[1]* B[2]=2*6=12,12 在结果中的位置是3和4. A[0]* B[2]=1*6=6, 6 在结果中的位置是2和3. 其它同理。 代码： Hits: 3

leetcode96 不同的二叉搜索树 Unique Binary Search Trees

Posted on 2019年11月22日2019年12月4日by jasonpeng

解法一 回溯法 求序列[1,…,n]的二叉树有多少种。 设d[i][j]为序列[i,..,j]的二叉树有多少种。 当i>j时，空树，1种； 当i==j时，单节点树，1种； 当i<j时，任选择一个 [i,..,j] 中的值x为根。 x左子树的种数是d[i][x-1]， x右子树的种数是d[x+1][j]。 x为根的总种数 = d[i][x-1] * d[x+1][j] 代码： 解法二 动态规划 对解法一分析，可发现区间[i,…,j]的种数，与区间的位置无关，而与区间的长度有关。 例如：d[1][4] 其实 等于 d[2][5]。 那么，直接以区间的长度为条件。 设d[n]表示n个节点的种数。 以第i (1 <= i <= n)个节点为根，将节点数分成两部分，一份是i-1个，一份是n-i个。 有：d[n] += d[i-1] * d[n-i] , 1<= i <= n 代码： Hits: 3
