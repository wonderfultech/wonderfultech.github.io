Posted on 2019年11月12日2019年12月4日by jasonpeng

解法一 滑动窗口法 设窗口的左边界为left. 初始值为-1. 设W为窗口中值的总和。初始值为0. 遍历到元素A[i]。 当W+A[i] >= s时，计算窗口长度i-left。收缩左边界left。 当W+A[i] < s时 ，窗口右边界向右滑动。 例如：s = 7, A = [2,3,1,2,4,3] left=-1,i=0. 当i=3时，窗口=[2,3,1]，W+A[3] = 6+2=8 > 7. 此时计算长度为4. 收缩左边界，窗口=[3,1]，W=4。 W+A[3]=4+2=6 < 7，窗口右边界向右滑动。 当i=4时，窗口=[3,1,2,4], W+A[4]=6+4=10 >7. 此时计算长度为4. 收缩左边界，窗口=[1,2,4]，W=3。 W+A[3]=3+4=6 >= 7，窗口长度为3. 如此下去。 完整代码： 代码： Hits: 5
