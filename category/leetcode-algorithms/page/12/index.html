leetcode108 将有序数组转换为二叉搜索树 Convert Sorted Array to Binary Search Tree

leetcode268 缺失数字 Missing Number

Posted on 2019年12月5日2019年12月5日by jasonpeng

解法一 交换法 本题与41. 缺失的第一个正数的解法 以及 287. 寻找重复数的解法相似。 当数字A[i] != i时，将A[i]与A[A[i]]交换。使得A[i] == i。 但是会有一个问题，当A[i]=N时，会越界，无法交换。此时只需跳出即可。 最后再遍历一遍A，A[i]!=i时，表示缺少i。 此外就缺少N. 例如：[3,0,1] A[0]=3,不交换A=[3,0,1] A[1]=0，交换A=[0,3,1]。 A[1]=3,不交换。A=[0,3,1] A[2]=1,交换A=[0,1,3]。 A[2]=3,不交换。A=[0,1,3] 缺少2. 例如：[2,0,1] A[0]=2，交换,A=[1,0,2]。 A[0]=1，交换，A=[0,1,2]。 缺的是3. 代码： Hits: 3

leetcode104 二叉树的最大深度 Maximum Depth of Binary Tree

Posted on 2019年12月5日2019年12月5日by jasonpeng

解法一 递归法 当root 为空节点时，深度为0. 当root不为空节点时，要在左右子树最大深度的最大值基础上，加上1个root节点。 左右子树的最大深度，有能递归计算。 例如： 3 / \ 9 20 / \ 15 7 叶子节点最大深度是1。 节点20，左右子树深度是1，其节点深度是max(1,1)+1=2。 节点3，左右子树深度是1和2，其节点深度是max(1,2)+1=3。 代码： Hits: 3
