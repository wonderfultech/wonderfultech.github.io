leetcode470 用 Rand7() 实现 Rand10() Implement Rand10() Using Rand7()

解法一 均匀分布 构造一个大于10的均匀分布. 然后从这个均匀分布中取值,并映射到10以内. 选取两个rand7的值构造一个二维的点阵. 点阵中共有49个点. 将最后9个点抛弃掉.然后将前40个点映射到10以内. 完整代码: 如果,第k次r 才小于等于40,前k-1次都失败. 设随机变量X表示首次取r就能小于等于40的实验次数. 概率p=40/49 X服从几何分布. P(X=k) = p(1-p)^(k-1) . 易知 P(X=k) 单调递减. X的期望=1/p=49/40=1.225. 这说明平均要1.2次才能得出r<=40. 因此while循环是必须要有的. 单次求出的值不精确. k=1,2,3,4,5时,P(X=k)的值,以及1-10的概率值. 可以看出k=1时,生成数字1到10的概率0.08 < 0.1. 随着k增大,概率值更小,与0.1相差的也更大. Hits: 6
