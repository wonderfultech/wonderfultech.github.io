leetcode292 Nim 游戏 Nim Game

解法一 找规律 谁面对4个石头，谁一定输。因为最多拿3个，剩余1个给对手。最少拿1个，剩余3个给对手。都是输。 谁面对8个石头，谁一定输。 因为最多拿3个，剩余5个给对手。对手拿1个，你再次面对4个石头，你输。 最少拿1个，剩余7个给对手。对手拿3个，你再次面对4个石头。你输。 其它为4的倍数，用同样的方法。 因此，当个数为4的倍数时，你一定输。 Hits: 3

leetcode189 旋转数组 Rotate Array

解法一 三次逆序 技巧很强的方法。代码好写，但是不容易想到。 首先k=0，没有移动任何元素。 如果k=数组的长度*x，x=0，1，2，3…，没有移动任何元素。 因此先处理k，k=k % 数组的长度 。 逆序方法： 1，序列尾部的k个元素逆序； 2，序列头部的N-k个元素逆序； 3，整个序列逆序； 例如：[1,2,3,4,5,6,7] 和 k = 3 尾部的[5,6,7]逆序成[7,6,5]. 头部的[1,2,3,4]逆序成[4,3,2,1]. 整个序列变成[4,3,2,1,7,6,5]再逆序成[5,6,7,1,2,3,4]。 代码： Hits: 3
