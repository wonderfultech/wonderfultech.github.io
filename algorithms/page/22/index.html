leetcode49 字母异位词分组 Group Anagrams

解法一 哈希表 字母相同但是位置不同。将每个字符串排序，排除掉字母顺序的影响。 将排序后结果相同的字符串用哈希表关联起来。 例如：[“eat”, “tea”, “tan”, “ate”, “nat”, “bat”] 对其中每个字符串排序。结果为：aet,aet,ant,aet,ant,abt。 那么aet关联到eat,tea,ate； ant关联到tan,nat abt关联到abt。 代码： Hits: 5

leetcode98 验证二叉搜索树 Validate Binary Search Tree

解法一 递归法 通过限定每个节点值的范围来验证二叉搜索树。 遍历节点root时，给出root值的范围(minv,maxv), minv < root.val < maxv 当root.val超出 范围(minv,maxv) 时，就不算二叉搜索树了。 递归左子树时，左子树的最大值不能超过root的值。 递归左子树时，右子树的最小值不能小于root的值。 例如： 根2的范围是(负无穷，正无穷)。 2的左子树1的范围是( 负无穷 ,2) 2的右子树3的范围是(2， 正无穷 ) 3的右子树4的范围是(3， 正无穷 ) 如果任何一个节点的值超过其应该的范围，一定不是二叉搜索树。 代码： Hits: 7

leetcode43 字符串相乘 Multiply Strings

解法一 竖式乘法 数字A与数字B相乘，A*B的总位数不会超过A和B的位数之和。 A[i] * B[j] 会贡献结果位置[i+j]和[i+j+1]上的值。 例如： num1 = “123”, num2 = “456” A[2] * B[2]= 3 * 6 =18, 18 在结果中的位置是4和5. A[1]* B[2]=2*6=12,12 在结果中的位置是3和4. A[0]* B[2]=1*6=6, 6 在结果中的位置是2和3. 其它同理。 代码： Hits: 3

leetcode64 最小路径和 Minimum Path Sum
