leetcode162 寻找峰值 Find Peak Element

解法一 二分查找法 本题关键在用好条件 A[i] ≠ A[i+1] 。 对于A[i]，A[i-1] ≠ A[i] ≠ A[i+1] 。 当A[i-1]>A[i]时，在i的左边可能会出现峰值。到左边接着找。 当A[i+1]>A[i]时，在i的右边可能会出现峰值。 到左边接着找。 其它情况，按照题目条件，A[i]就是峰值了。 完整思路： 代码： Hits: 5

leetcode1115 交替打印FooBar Print FooBar Alternately

解法一 信号量法 foo和bar交替输出。这种线程间协调，是信号量的典型应用。 设计信号量foo_m=1,bar_m=0。 foo_m=1表示foo先输出。bar=0表示bar等着。 当foo输出后，将bar设置为1，之后bar才能输出。 bar输出之后，将foo_m置为1，foo才能再次输出。 完整思路： 代码： Hits: 3

leetcode153 寻找旋转排序数组中的最小值 Find Minimum in Rotated Sorted Array
