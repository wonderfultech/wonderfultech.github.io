leetcode415 字符串相加 Add Strings

解法一 按位加法 首先明确数字串是高位数字放在底索引处. “123”. 1是高位数字. 整数相加, 因为有进位, 从数字的低位数字相加, 再向高位数字进位. 也就是索引要从高到低遍历串. 例如: “123” + “7” 先3+7=10,进位1,值为0. 再2+1=3, 进位0,值为3 再1+0=1,进位0,值为1. 结果是130. 代码: Hits: 3

leetcode543 二叉树的直径 Diameter of Binary Tree

解法一 后序遍历法 空节点高度为0. 叶节点高度为1. 过根节点A的直径 = 左子树的高度 + 右子树的高度。 左右子树的高度递归求解。同时A节点的高度 = 左右子树高度的最大值 + 1. 代码： Hits: 4

leetcode6 Z 字形变换 ZigZag Conversion

解法一 偏移法 计算每行各个字符在原串中的偏移。 两长列之间的偏移 = 行数+行数-2. 两长列之间斜线上的字符偏移 = 两长列之间的偏移 – 2 * 行号。 例如：s = “LEETCODEISHIRING”, numRows = 4 行数是4。 从图形上观察两长列之间的斜线上还有两个字符。 因此偏移是4+4-2=6. 斜线上的字符偏移 = 6-2 * 行号 第一行的字符偏移0，6，12. 第二行的字符偏移1，7，13. 斜线上的字符偏移 = 6-2 * 1 = 4 两长列间斜线上的字符偏移是1+4=5,7+4=11,13+4=17。 第三行的字符偏移2，8，14. 斜线上的字符偏移 = 6-2 * 2 = 2 两长列间斜线上的字符偏移是2+2=4, 8+2=10, 14+2=16。 代码： Hits: 3

leetcode16 最接近的三数之和 3Sum Closest

解法一 排序法 解法套用 15. 三数之和 的解法框架。 先对序列排序。选择A[i]为第一个数，再区间[i+1,N-1]中选择两个数。 设选择A[x]和A[y]，计算值s=A[i]+A[x]+A[y]。用s更新结果。 当s > target时，s要变小。y向左移动。 当s < target时，s要变大。x向右移动。 当s == target时，恰好。 代码： Hits: 4

leetcode109 有序链表转换二叉搜索树 Convert Sorted List to Binary Search Tree

解法一 二分法 构造平衡树。对一个链表，每次选在中间节点作为根节点。中间节点前后的链表分别递归构造两棵平衡树。 关键是正确的将链表切分成两半。 例如：[-10,-3,0,5,9] 要选择0为根，[-10,-3]为左子树, [5,9]为右子树。例如： [-10,-3,0,5,9,10] 要选择5为根，[-10,-3，0]为左子树, [9，10]为右子树。 链表切分思路： 代码： 将后一半链表的第一个节点作为根节点。并对左右边的链表，递归构建二叉树。 代码： Hits: 6
