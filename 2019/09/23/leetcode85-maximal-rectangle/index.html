leetcode85 最大矩形 Maximal Rectangle

Posted on 2019年9月23日2019年12月4日by jasonpeng

leetcode 数据库题目全部题解

本题在于找规律。

Table of Contents

  • 解法一 枚举法
  • 解法二 单调队列

解法一 枚举法

本题的时间卡不紧，故而本方法可行。

先找规律。以题目中的case为例。

考虑位置(2,3)出的最大矩形。

一种是向左矩形，从位置(2,3)向左延申，红色框圈住的矩形；高度是1，宽度是向左连续1的个数。

一种是向上矩形，从位置(2,3)向上延申，绿色框圈住的矩形；宽度是1，高度是向上连续1的个数。

再一种是， 以位置(2,3)为右下角时，紫色框圈住的矩形；

如此，可发现第一个规律，每个位置处都有这三个情况，最大矩形就从这三个矩形中选出。

由于位置(2,2)值为1， 位置(2,3) 也为1。因此在计算 位置(2,3) 处向左的矩形时，可在 位置(2,2) 的向左矩形基础上扩展即可，无需从头算。

同样， 位置(1,3)为1， 位置(2,3) 向上的矩形 可在 位置(1,3) 向上矩形的基础上扩展，也无需从头算。

可得出，第二个规律，每个位置向左和向上的矩形，存在递推关系。

而对于以 位置(i,j) 为右下角的矩形。

最简单的方法，从行i向上遍历，比如遍历到行k， 行k与行i之间不能有长度为0的向左矩形。行k与行i之间，由每个位置处的向左矩形，就组成了一个区域。如下图。

从这个区域中，求位置(i,j)的最大矩形。关键是找出最窄的向左矩形。最大矩形的面积 = 最短向左矩形的宽度 x (i-k+1)。

另外，由向上矩形的定义看，i-k+1 == 向上矩形的高度。

那么，本题的整体思路：

vvv[i][j][0]: 位置(i,j)处向左矩形的宽度;
vvv[i][j][1]: 位置(i,j)处向上矩形的高度;
res:最大矩形的面积；
for(行i){
    for(列j){
        if(位置(i,j)值为0) 矩形面积为0，不用处理,continue；
        //向左矩形
        if(位置(i,j-1)值为1) vvv[i][j][0]=max(1,vvv[i][j-1][0]+1);
        //向上矩形
        if(位置(i-1,j)值为1) vvv[i][j][1]=max(1,vvv[i-1][j][1]+1);
        //以位置(i,j)为右下角的矩形
        宽度len=vvv[i][j][0];
        for(k=i;k>i-vvv[i][j][1];--k){
            len = min(len,vvv[k][j][0]);
            res = max(res,len * (i-k+1));
        }
        res = max(res, max(vvv[i][j][0],vvv[i][j][1]));
    }
}

AC的代码：

    int R,C;
    int maximalRectangle(vector<vector<char>>& matrix) {
        if(!(R=matrix.size()) || !(C=matrix[0].size())) return 0;
        vector<vector<vector<int>>> vvv(R,vector<vector<int>>(C,vector<int>(2,0)));
        const int cnt=2;
        const static int dx[cnt]={0,-1};
        const static int dy[cnt]={-1,0};
        int res = 0;
        for(int i=0;i<R;++i){
            for(int j=0;j<C;++j){
                if(matrix[i][j] == '0') continue;
                vector<int>& v = vvv[i][j];
                v[0]=v[1]=1;
                for(int k=0;k<cnt;++k){
                    const int ni = i + dx[k];
                    const int nj = j + dy[k];
                    if(!in_grid(ni,nj)) continue;
                    v[k] =  max(v[k],vvv[ni][nj][k] + 1);
                }
                int len=v[0];
                for(int up=i;up > i-v[1];--up){
                    len = min(len,vvv[up][j][0]);
                    res = max(res,len * (i-up+1));
                }
                res = max(res,max(v[0],v[1]));
            }
        }
        return res;
    }
    
    bool in_grid(int x,int y){
        return x>=0 && x<R && y>=0 && y <C;
    }

解法二 单调队列

其实，是对解法一的改进。

解法一，计算以位置(i,j)为右下角的矩形时，枚举矩形，再求面积最大的矩形。

再看这个图。

对矩阵的第j列，从上往下看，将问题变成，求一维数列上，面积最大的矩形。

这样，对每列求最大矩形方法，与下图中从左到右， 求最大黄色块矩形的方法相同。

只不过，本题中i的值是不断增加的。

求最大黄色块矩形，就是leetcode84题，柱状图中最大的矩形。这里不深入讨论84题，这里直接给出解决思路。

对这个问题，可以用单调队列来解。队列中放每个高度的位置。

用单调队列求最大黄色块矩形的思路：

数组A,A[i]是位置i上的高度;
队列Q,队列中放位置i;
数组A尾部补上个0;
res 最大矩形面积;
for(i in A){
    //A[i]入队列不能形成单调增加队列，所以出栈求面积。
    while(Q不为空 且 A[Q尾部值] > A[i]){
        a = Q尾部值;
        Q尾部弹出;
        if (Q不为空){
            b = Q尾部值;
        }else{
            b = -1;
        }
        //求面积
        res = max(res,A[a] * (i - b - 1));
    }
    i入Q尾部;
}

要注意的是，队列Q中，从头到尾，位置的高度，是单调增加的。当够不成单调队列时，要出队列，求面积。数组A尾部放个0，便于求完整的面积。

由于只在队列的尾部进行操作，也可以用栈代替队列。

回到本题，对矩阵的每一列j，用一个队列Q[j]表示。每算出一个位置的向左矩形的宽度h，将h加入队列Q[j]。

完整思路：

vvv[i][j]: 位置(i,j)处向左矩形的宽度;
vv[j]: 列j处的单调队列Q;
res:最大矩形的面积；
for(行i){
    for(列j){
        if(位置(i,j)值为1){ 
            //向左矩形
            if(位置(i,j-1)值为1) vvv[i][j]=max(1,vvv[i][j-1]+1);    
        }
        //以位置(i,j)为右下角的矩形
        add_left(vvv,vvv,i,j,vvv[i][j]);
        if(i == R-1 && vvv[i][j]){
            add_left(vvv,vvv,R,j,0);
        }
    }
}

将向左矩形宽度添加到队列Q的思路：

res 最大矩形面积;
add_left(vvv,vv,i,j,w){
    队列Q = vv[j];
    while(Q不为空 且 vvv[Q尾部值][j] > w){
        a = Q尾部值;
        Q尾部弹出;
        if (Q不为空){
            b = Q尾部值;
        }else{
            b = -1;
        }
        //求面积
        res = max(res,vvv[a][j] * (i - b - 1));
    }
    i入Q尾部;    
}

完整的代码：

    int R,C;
    int res=0;
    int maximalRectangle(vector<vector<char>>& matrix) {
        if(!(R=matrix.size()) || !(C=matrix[0].size())) return 0;
        vector<vector<int>> vvv(R,vector<int>(C,0));
        vector<vector<int>> vv(C,vector<int>());
        res = 0;
        for(int i=0;i<R;++i){
            for(int j=0;j<C;++j){
                int& ans = vvv[i][j];
                if(matrix[i][j] == '1') {
                    ans=1;
                    const int ni = i;
                    const int nj = j - 1;
                    if(in_grid(ni,nj) && matrix[ni][nj] == '1'){
                        ans = max(ans,vvv[ni][nj] + 1);
                    }
                }
                add_left(vvv,vv,i,j,ans);
                if(i == R-1 && ans){
                    add_left(vvv,vv,R,j,0);
                }
            }
        }

        return res;
    }
    
    void add_left(vector<vector<int>> & vvv,
                  vector<vector<int>> & vv,
                  int i,int j,int h){
        vector<int>& Q=vv[j];
        while(Q.size() && vvv[Q.back()][j] > h){
            int a = Q.back();Q.pop_back();
            int b = Q.size() ? Q.back() : -1;
            res = max(res,vvv[a][j] * (i - b - 1));
        }
        Q.push_back(i);
    }
