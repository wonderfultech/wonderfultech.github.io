leetcode10 正则表达式匹配 Regular Expression Matching

解法一 回溯法

设s(i)表示串s从位置i开始的后缀。p(j)同理。

直观的思路是对串s和p分情况讨论。

1.当s为空串时，此时p为空串，匹配成功。

如果p不为空时，

例如：s="",p="a"

匹配失败。

例如：s="",p="a*"

匹配成功。

例如：s="",p="a*c"

匹配又失败。

说明，当p的第二个字符，即p[1]=='*'时，要递归判断s和p(2)。

2.当p为空时，此时s不为空，匹配失败。

例如：s="a", p =""。

匹配失败。

3.如果s和p的第一个字符匹配时，即s[0] == p[0] 或 p[0] == '.'时，要对除第一个字符外的s(1)和p(1)递归匹配。

例如：s="abc", p =".*d"

第一字符匹配。整体结果还要考虑"bc"和"*d"的匹配情况。

4.当p[1] == '*'时，

如果*与其前一个字符匹配s的0个字符时，相当于p少了这两个字符，要递归考虑s和p(2)。

例如： s="abc", p =".*d"

".*"匹配0个字符，要递归考虑s="abc"和p(2)="d"。

如果*与其前一个字符匹配s的k个字符时，相当于s少了这k个字符，而p少了这两个字符，要递归考虑s(k)和p(2)。

例如： s="aaacdgg"，p="a*ceg"。

"a*"可以匹配1个a，此时递归考虑s="aacdgg"和p="ceg"。

"a*"可以匹配2个a，此时递归考虑s="acdgg"和p="ceg"。

"a*"可以匹配3个a，此时递归考虑s="cdgg"和p="ceg"。

5.其它情况，匹配失败。

完整的思路：

dfs(s,p){
    //情况1
    if(s空串) 返回：p空串与否 或者 p.length>=2 且 p[1] == '*' 且 dfs(s,p(2));
    //情况2
    if(p空串) 返回：s空串与否；
    //情况3
    if(s[0] == p[0] 或 p[0] == '.') 递归dfs(s(1),p(1));
    //情况4
    if(p.length>=2 且 p[1]=='*'){
        //匹配0个字符
        递归dfs(s,p(2));
        //匹配c个字符
        递归dfs(s(c),p(2));
    }
}

完整的代码：

    bool isMatch(string s, string p) {
        return dfs(s,p);
    }
    
    bool dfs(string s,string p){
        if(s.empty()) return p.empty() || p.size() >= 2 && p[1] == '*' && dfs(s,p.substr(2));
        if(p.empty()) return false;
        if(s[0] == p[0] || p[0] == '.') {
            bool r = dfs(s.substr(1),p.substr(1));
            if(r) return true;
        }
        if(!(p.size() >= 2 && p[1] == '*')) return false;
        bool r = dfs(s,p.substr(2));
        if(r) return true;
        for(int c=0;c<s.size();++c){
            if(s[c] == p[0] || p[0] == '.') {
                bool r = dfs(s.substr(c+1),p.substr(2));
                if(r) return true;
            }else break;
        }
        
        return false;
    }

简化

在上面的解法中，p[1]=='*'分散在两处，而且*匹配所起的作用非常重要。

将*匹配的整合在一起。其它不变。

调整思路为：

dfs(s,p){
    //情况1
    if(s空串 且 p空串) 返回成功;
    //情况4
    if(p.length>=2 且 p[1]=='*'){
        //匹配0个字符
        递归dfs(s,p(2));
        //匹配c个字符
        递归dfs(s(c),p(2));
    }
    //情况3
    if(s[0] == p[0] 或 p[0] == '.') 递归dfs(s(1),p(1));
}

代码为：

    bool isMatch(string s, string p) {
        return dfs(s,p);
    }
    
    bool dfs(string s,string p){
        if(s.empty() && p.empty()) return true;
        if(p.size() >= 2 && p[1] == '*'){
            bool r = dfs(s,p.substr(2));
            if(r) return true;
            for(int c=0;c<s.size();++c){
                if(s[c] == p[0] || p[0] == '.') {
                    bool r = dfs(s.substr(c+1),p.substr(2));
                    if(r) return true;
                }else break;
            }
        }
        if(s.size() && p.size() && (s[0] == p[0] || p[0] == '.')) {
            bool r = dfs(s.substr(1),p.substr(1));
            if(r) return true;
        }        
        return false;
    }

解法二 动态规划

设s(i)表示串s从位置i开始的后缀。p(j)同理。

slen是串s的长度。plen是串p的长度。

表格d[i][j]表示s(i)和p(j)的匹配情况。

回溯法，实际是在递归中先判断后缀是否匹配。再得出当前位置是否匹配。随着后缀的增长，直到整个串都匹配。

从后向前递推

将后缀串匹配的思路，由递归方式变成递推方式。

当s(slen)且p(plen)时，两个都是空串，匹配成功。d[slen][plen]=1。

当s(slen) ,p(j), j< plen时，s后缀是空串，但是p(j)不是空串。

此时，只有当p[j+1] == '*' 并且 s(slen)与p(j+2)匹配时，才能匹配成功。好在， s(slen)与p(j+2)匹配情况已经知道。

即d[slen][j] = (p[j+1]=='*' 且 d[slen][j+2])。

当s(i),i<slen,p(plen)时，p后缀是空串，匹配失败。d[i][plen]=0。

当s(i),i<slen,p(j),j<plen时，s后缀和p后缀都不是空串时。

也要分几种情况考虑。

当s[i] == p[j] 或 p[j] == '.'时，要考虑 s(i+1)与p(j+1)的情况。

d[i][j] = (s[i] == p[j] 或 p[j] == '.' ) 且 d[i+1][j+1]。

当p[j+1]=='*'时，要考虑*与p[j]匹配s(i)的0个字符的情况。

d[i][j] = d[i][j+2];

当p[j+1]=='*'时，要考虑*与p[j]匹配s(i)的c个字符的情况。

d[i][j] = (s[c] == p[j] || p[j]=='.') 且 d[c+1][j+2]。

可以看出思路与回溯法完全相同。

在转成代码时，还要考虑边界条件。

完整的代码：

    bool isMatch(string s, string p) {
        const int slen = s.size();
        const int plen = p.size();
        vector<vector<int>> d(slen+1,vector<int>(plen+1,0));
        for(int i=slen;i>=0;--i){
            for(int j=plen;j>=0;--j){
                if(i == slen && j == plen){
                    d[i][j] = 1;
                }else if(i == slen && j < plen){
                    d[i][j] = j+1 < plen && p[j+1] == '*' && j+2 <= plen && d[i][j+2];
                }else if(i < slen && j == plen){
                    d[i][j] = 0;
                }else{
                    d[i][j] = (s[i] == p[j] || p[j] == '.') && d[i+1][j+1];
                    if(j + 1 < plen && p[j + 1] == '*' && j + 2 <= plen){
                        d[i][j] = d[i][j] || d[i][j+2];
                        for(int c = i;c < slen && (s[c] == p[j] || p[j] == '.'); ++c) {
                              d[i][j] = d[i][j] || d[c+1][j+2];
                        }
                    }
                }
            }
        }
        return d[0][0];
    }

从前往后递推

设s(i)表示长度为i的前缀串。p(i)同理。

s(i)最后一个元素是s[i-1]。同理p[j-1]。

表格d[i][j]表示s(i)和p(j)的匹配情况。

分情况讨论。

当s(0)，与p(0)时，两个空串。匹配成功。d[0][0]=1。

当s(0),p(j),j>0时，此时s(0)空串，p(j)非空。

只有当p[j-1] == '*'并且s(0)与p(j-2)匹配时，才能匹配成功。

d[0][j] = (p[j-1] == '*') 且 d[0][j-2]。

当s(i),i>0,p(0)时，此时p(0)空串，s(i)非空。匹配失败。

d[i][0]=0。

当s(i),i>0,p(j),j>0时，分情况讨论。

当s[i-1] == p[j-1] 或 p[j-1] =='.'时，s(i-1)和p(j-1)匹配与否，起决定作用。

d[i][j]= ( s[i-1] == p[j-1] 或 p[j-1] =='.' ) 且 d[i-1][j-1]。

当p[j-1]=='*'时，*与p[j-2]同s(i)匹配0个字符的情况。此时*与其前一个字符相当于不存在。

d[i][j] = d[i][j-2]。

当p[j-1]=='*'时，*与 p[j-2]同 s(i)匹配c个字符的情况。

d[i][j] = (s[i-c] == p[j-2] 或 p[j-2] == '.') 且d[i-c][j-2]。

一定要注意，i-c在s[i-c]和在d[i-c][j-2]中的区别。

前者表示s串中i-c 位置 。后缀表示前缀串的长度为i-c。

正因为这一点，相比于从后往前推，容易写错。

最终结果是d[slen][plen]。

完整的代码：

    bool isMatch(string s, string p) {
        const int slen = s.size();
        const int plen = p.size();
        vector<vector<int>> d(slen+1,vector<int>(plen+1,0));
        for(int i=0;i<=slen;++i){
            for(int j=0;j<=plen;++j){
                if(i == 0 && j == 0){
                    d[i][j] = 1;
                }else if(i == 0 && j > 0){
                    d[i][j] = j>=2 && (p[j-1] == '*') && d[i][j-2];
                }else if(i >0 && j == 0){
                    d[i][j] = 0;
                }else{
                    d[i][j] = (s[i-1] == p[j-1] || p[j-1] =='.') && d[i-1][j-1];
                    if(p[j-1] == '*' && j>=2){
                        d[i][j] = d[i][j] || d[i][j-2];
                        for(int c=1;c <= i && (s[i - c] == p[j-2] || p[j-2] == '.');++c){
                            d[i][j] = d[i][j] || d[i - c][j-2];
                        }
                    }
                }
            }
        }
        return d[slen][plen];
    }
