linear regression 线性回归

Posted on 2019年8月6日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

在已知自变量和因变量的观察数据的情况下，用线性回归求解自变量和因变量之间的线性方程。

线性回归易于理解和编程。非常适合用来练手。

单自变量线性回归

已知自变量\(x\)和因变量\(y\)的观察数据：

41 90
42 93
43 98
20 64
25 78
40 71
58 88
60 86

假设两者之间满足公式:

$$ y = ax + b $$

由于每对观测数据都满足上面的公式。

共有\(N\)组数据，可得出:

\begin{align} y_1 & = ax_1 + b \\ y_2 & = ax_2 + b \\ & \dots \\ y_N & = ax_N + b \end{align}

除此之外，损失函数（\(y\)的预测值与 \(y\) 的观测值之间的误差）平方和最小。

$$ \epsilon = \sum_{i=1}^N \left( ax_i + b - y_i \right) ^ 2 $$

那么，\(\epsilon\)可以认为是关于自变量\(a\)和 \(b\) 的函数。要求 \(\epsilon\) 最小，等价于最优化问题：

$$ min \space \epsilon = \sum_{i=1}^N \left( ax_i + b - y_i \right) ^ 2 \label{epsilon_def_ref} $$

用初等数学对公式\eqref{epsilon_def_ref}展开得：

\begin{align} \epsilon & = \sum_{i=1}^N \left( ax_i + b - y_i \right) ^ 2 \\ & = a^2 \sum_{i=1}^N x_i^2 + nb^2 + \sum_{i=1}^N y_i^2 + 2ab \sum_{i=1}^N x_i - 2a \sum_{i=1}^N x_i y_i -2b \sum_{i=1}^N y_i \end{align}

对于无约束最优化问题 \eqref{epsilon_def_ref} ，寻找其最小值点。求其极值点。求其关于\(a\)和 \(b\)的偏导，并令其等于0。

\begin{align} \frac {\partial \epsilon} {\partial a} & = 2a \sum_{i=1}^Nx_i^2 + 2b \sum_{i=1}^N x_i -2 \sum_{i=1}^Nx_i y_i = 0 \\ \frac {\partial \epsilon} {\partial b} & = 2nb + 2a \sum_{i=1}^N x_i -2 \sum_{i=1}^N y_i = 0 \end{align}

两边整理得：

\begin{align} a \left( \sum_{i=1}^N x_i^2 \right) + b \left( \sum_{i=1}^N x_i \right) &= \sum_{i=1}^N x_i y_i \\ a \left( \sum_{i=1}^N x_i \right) + bn & = \sum_{i=1}^N y_i \end{align}

只要满足系数行列式不为0，克莱默法则成立，就可快速求出 \(a\)和 \(b\) 。

系数行列式为：

\begin{align} \begin{vmatrix} \left( \sum_{i=1}^N x_i^2 \right) & \left( \sum_{i=1}^N x_i \right) \\ & & \\ \left( \sum_{i=1}^N x_i \right) & n \end{vmatrix} & = n\left( \sum_{i=1}^N x_i^2 \right) - \left( \sum_{i=1}^N x_i \right)^2 \end{align}

由克莱默法则，得出结果为:

\begin{align} a & = \frac {n\left( \sum_{i=1}^N x_i y_i \right) - \left( \sum_{i=1}^N y_i \right)\left( \sum_{i=1}^N x_i \right) } {n\left( \sum_{i=1}^N x_i^2 \right) - \left( \sum_{i=1}^N x_i \right)^2} \\ b & = \frac {\left( \sum_{i=1}^N x_i y_i \right)\left( \sum_{i=1}^N x_i \right) - \left( \sum_{i=1}^N y_i \right)\left( \sum_{i=1}^N x_i^2 \right) } {\left( \sum_{i=1}^N x_i \right)^2 - n\left( \sum_{i=1}^N x_i^2 \right) } \end{align}

从上面的公式中可以看出。关键是求出一系列的和。

$$ \sum_{i=1}^N x_i , \sum_{i=1}^N y_i, \sum_{i=1}^N x_i^2, \sum_{i=1}^N x_i y_i $$

为单自变量设计mapreduce任务

map.py

发射x,y,xy和x^2即可。

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    segs = line.split()
    x = long(segs[0])
    y = long(segs[1])
    xy = x*y
    xx = x*x
    print('%d\t%d\t%d\t%d'%(x,y,xy,xx))

reduce.py

累加x,y,xy和x^2的值。

#!/usr/bin/python
#coding=utf-8
import sys

x_sum = 0
y_sum = 0
xy_sum = 0
xx_sum = 0
N = 0

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    x = long(segs[0])
    y = long(segs[1])
    xy = long(segs[2])
    xx = long(segs[3])
    x_sum += x
    y_sum += y
    xy_sum += xy
    xx_sum += xx
    N += 1

def gen_final_results():
    '''
    '''
    T = N * xx_sum - x_sum * x_sum
    if T == 0:
        print('无唯一解')
        return
    A = N * xy_sum - x_sum * y_sum
    B = y_sum * xx_sum - x_sum * xy_sum
    a = A / float(T)
    b = B / float(T)
    print('%s\t%s' % (a,b))

gen_final_results()

多变量线性回归

寻找多个自变量\(x_1,x_2,\dots,x_n\)和单个因变量\(y\)之间的线性关系。其中\(N\)为自变量个数。

已有观察数据：每行11列，前10为个自变量，最后一个为因变量。

3.807590643342410180e-02 5.068011873981870252e-02 6.169620651868849837e-02 2.187235499495579841e-02 -4.422349842444640161e-02 -3.482076283769860309e-02 -4.340084565202689815e-02 -2.592261998182820038e-03 1.990842087631829876e-02 -1.764612515980519894e-02 1.510000000000000000e+02
-1.882016527791040067e-03 -4.464163650698899782e-02 -5.147406123880610140e-02 -2.632783471735180084e-02 -8.448724111216979540e-03 -1.916333974822199970e-02 7.441156407875940126e-02 -3.949338287409189657e-02 -6.832974362442149896e-02 -9.220404962683000083e-02 7.500000000000000000e+01
......

假设自变量\(x_1,x_2,\dots,x_n\)和单个因变量\(y\)之间的线性关系：

$$ y = a_1x_1 + a_2x_2+\dots+a_nx_n + b $$

假设有\(m\)行数据。则有方程组：

\begin{align} y_1 & = a_1x_{11} + a_2x_{12}+\dots+a_nx_{1n} + b \\ y_2 & = a_1x_{21} + a_2x_{22}+\dots+a_nx_{2n} + b \\ & \dots \\ y_i & = a_1x_{i1} + a_2x_{i2}+\dots+a_nx_{in} + b \\ & \dots \\ y_m & = a_1x_{m1} + a_2x_{m2}+\dots+a_nx_{mn} + b \end{align}

为方便，可以记:

\( X_i = \left( x_{i1},x_{i2},\dots,x_{in},1 \right) \)

$$ X = \begin{bmatrix} X_1 \\ X_2 \\ \dots \\ X_m \end{bmatrix} $$

\( A = \left( a_1,a_2,\dots,a_n,b \right)^T \)

\( Y = \left( y_1,y_2,\dots,y_m \right)^T \)

借助\( X_i \)和\( A \)，上述方程组和简化为：

$$ Y = XA $$

损失函数（\(Y\)的预测值与 \(Y\) 的观测值之间的误差）平方和最小。

即求最优化问题:

$$ min \space \epsilon = \sum_{i=1}^m \left( X_i A -y_i \right)^2 $$

对公式进行化简得出：

\begin{align} \epsilon & = \sum_{i=1}^m \left( X_i A -y_i \right)^2 \\ & = \left( X_1A-y_1,X_2A-y_2,\dots,X_mA-y_m \right) \begin{bmatrix} X_1A-y_1 \\ X_2A-y_2 \\ \dots \\ X_mA-y_m \end{bmatrix} \\ & = (XA-Y)^T(XA-Y) \\ & = (A^TX^T-Y^T)(XA-Y) \\ & = A^TX^TXA - A^TX^TY-Y^TXA-Y^TY \end{align}

求误差关于\(A\)的偏导，并令偏导等于0。

\begin{align} \frac {\partial \epsilon }{\partial A} &=2X^TXA-2X^TY \label{pa_ref} \\ & = 0 \end{align}

因此只要\(X^TX)\)可逆就能求出\(A\)的唯一解。

从公式\eqref{pa_ref}知，需要求出\(X^TX\)和\(X^TY\)。

求解\(X^TX\)和\(X^TY\)

\begin{align} X^TX &= \left(X_1^T,X_2^T,\dots,X_m^T \right) \begin{bmatrix} X_1 \\ X_2 \\ \dots \\ X_m \end{bmatrix} \\ & = X_1^TX_1 + X_2^TX_2+\dots+X_m^TX_m \\ & = \sum_{i=1}^mX_i^TX_i \end{align}

其中\( X_i^TX_i \)非常关键：

\begin{align} X_i^TX_i & = \begin{bmatrix} X_{i1} \\ X_{i2} \\ \dots \\ X_{in} \\ 1 \end{bmatrix} \left( X_{i1},X_{i2},\dots,X_{in},1 \right) \\ & = \begin{bmatrix} X_{i1}X_{i1} \space X_{i1}X_{i2} \space \dots \space X_{i1}X_{in} \space X_{i1} \\ X_{i2}X_{i1} \space X_{i2}X_{i2} \space \dots \space X_{i2}X_{in} \space X_{i2} \\ \vdots \\ X_{in}X_{i1} \space X_{in}X_{i2} \space \dots \space X_{in}X_{in} \space X_{in} \\ X_{i1} \space \space X_{i2} \space\space \dots \space X_{in} \space \space 1 \label{xixi_ref} \end{bmatrix} \end{align}

从公式\eqref{xixi_ref}中可看出，每读入一行数据。按照第i个自变量乘以该组自变量就可得出矩阵的第i行。

\(X^TY\)的计算结果如下：

\begin{align} X^TY &=\left( X_1^T,X_2^T,\dots,X_m^T \right) \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_m \end{bmatrix} \end{align}

因此，每读入一行数据。将因变量乘以每行的自变量即可。

为多变量设计mapreduce任务

map2.py

每读入一行数据，发射\( X_i^TX_i \)和\( y_iX_i^T \)

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    segs = line.split()
    #X矩阵的每一行
    Xi = [float(d) for d in segs[:-1]]
    Xi.append(1)
    Y = float(segs[-1])
    #计算Xi'Xi，
    res_XiXi = ''
    for x in Xi:
        y = [str(x*d) for d in Xi]
        res_XiXi += ' '.join(y) + ' ; '
    print(res_XiXi)
    #计算Y'Xi
    res_YXi = ''
    for x in Xi:
        y = [str(Y*d) for d in Xi]
        res_YXi += ' '.join(y) + ' , '
    print(res_YXi)

reduce2.py

累加 \( X_i^TX_i \)和\( y_iX_i^T \) 以得出\(X^TX\)和\(X^TY\)。

之后判断\(X^TX\)是否可逆，如果可逆，就有唯一解，用numpy求解。

$$ A = \left( X^TX \right)^{-1} \left( X^TY \right) $$

如果不可逆，就无唯一解。需要通过其它方法求解，本文不支持。

#!/usr/bin/python
#coding=utf-8
import sys

#加入numpy的搜索路径
sys.path.append("~/anaconda2/lib/python2.7/site-packages")

import numpy as np

X_plus_X = []
X_plus_Y = []

def add_on_XX(line,XX):
    '''
    添加到X'X矩阵上
    '''
    segs = line.split(';')
    for i,row in enumerate(segs):
        if row == '':
            continue
        col = row.split()
        if i == len(XX):
            XX.append([0 for j in range(len(col))])
        for j,c in enumerate(col):
            XX[i][j] += float(c)

def add_on_YX(line,XY):
    '''
    添加到X'Y矩阵上。mapper传过来的是转置值。
    '''
    segs = line.split(',')
    for i,row in enumerate(segs):
        if row == '':
            continue
        col = row.split()
        if len(XY) == 0:
            XY.append([0 for j in range(len(col))])
        for j,c in enumerate(col):
            XY[0][j] += float(c)
             

for line in sys.stdin:
    line = line.strip()
    #print(line)
    if ';' in line:
        #值是Xi'Xi
        add_on_XX(line,X_plus_X)
    elif ',' in line:
        #值是Y'Xi
        add_on_YX(line,X_plus_Y)
    else:
        continue

def result():
    '''
    '''
    #print(X_plus_X)
    #print(X_plus_Y)

    #转成numpy数组，方便求逆
    XX = np.array(X_plus_X)
    XY = np.array(X_plus_Y)

    #先判断X'X矩阵是否可逆
    if np.linalg.det(XX) == 0:
        print('不可逆，没有唯一解')
    else:
        XX_inv = np.linalg.inv(XX)
        XY_plus = XY.transpose()
        A = np.dot(XX_inv,XY_plus)
        print(A)

result()
