find all triangles in the graph 发现图中所有的三角形

Posted on 2019年8月3日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

一个有趣的问题。现有一张无向图。要统计出该图中所有的三角形。乍一看，用mapreduce框架解决，此问题无思路。

三角形一定有三条边，三个顶点构成。设顶点按顺时针依次为a，b，c。

此时，就有三条边，a-b,b-c,c-a。

假设现在不知道，a，b，c三点能形成三角形。但是知道存在边a-b和a-c。

那么只要b和c直接相连形成边，就能断定a，b，c能形成三角形。

以此为突破口，设计如下的算法流程：

Table of Contents

  • 三角形形成算法
  • 设计mapreduce任务

三角形形成算法

发射边：

  1. 无向图中每个顶点关联的所有边都发射出去。发射(a,b)。
  2. 相同起点的边，收集起来。(a,b1),(a,b2),...,(a,bn)。
  3. 那么可知，b1,b2,...,bn之间，肯定通过顶点a相关联。因此， b1,b2,...,bn之间可能形成边。但是不确定，所以发射(bi,bj,a)。只要之后碰到(bi,bj)的直接边就能形成三角形。而已知的直接边，发射成(a,bi,-)。'-'相当于直接边的标记。

组合三角形

  1. 有两个相同点的边，收集起来。(a,b,-),(a,b,c1), (a,b,c2), ..., (a,b,cn)。
  2. 既有直接边，又有非直接边，的情形才能形成三角形。即，-,c1,c2,...,cn的形式。'-'一定要有，ci至少有一个。
  3. 发射三角行。符合条件2，就可形成三角形:(a,b,c1), (a,b,c2), ..., (a,b,cn)。

三角形去重

去重较简单。对三角形的三个顶点排序。借助mapreduce框架的排序功能。然后去重。

设计mapreduce任务

为上面的三个阶段，每个都设计一个mr任务。

发射边：

map.py

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    segs = line.split()
    print('%s,%s' % (segs[0],segs[1])) 
    print('%s,%s' % (segs[1],segs[0]))

reduce.py

#!/usr/bin/python
#coding=utf-8
import sys

last_key = ''
last_value = []

def gen(key,values):
    '''
    '''
    for x in values:
        print('%s,%s,%s'%(key,x,'-'))

    for i,x in enumerate(values):
        for y in values[i+1:]:
            print('%s,%s,%s'%(x,y,key))

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    key = segs[0]
    value = segs[1]
    if last_key != key:
        if last_key != '':
            gen(last_key,last_value)
            last_key = ''
            last_value = []
    last_key = key
    last_value.append(value)

if last_key != '':
    gen(last_key,last_value)

reduce2.py

#!/usr/bin/python
#coding=utf-8
import sys

last_key = ''
last_value = []

def gen(key,values):
    '''
    '''
    if '-' not in values:
        return
    for x in values:
        if x == '-':
            continue
        print('%s,%s' % (key,x))

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    key = segs[0]
    value = segs[1]
    if last_key != key:
        if last_key != '':
            gen(last_key,last_value)
            last_key = ''
            last_value = []
    last_key = key
    last_value.append(value)

if last_key != '':
    gen(last_key,last_value)

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    #print(line) 
    segs = line.split(',')
    segs.sort()
    print(','.join(segs))

reduce3.py

#!/usr/bin/python
#coding=utf-8
import sys

last_key = ''
last_value = []

def gen(key):
    '''
    '''
    print(key)

for line in sys.stdin:
    line = line.strip()
    #print(line)
    key = line
    if last_key != key:
        if last_key != '':
            gen(last_key)
            last_key = ''
    last_key = key

if last_key != '':
    gen(last_key)

提交脚本：

#!/bin/bash
#hadoop 2.7.6
JAR=${HADOOP_HOME}/share/hadoop/tools/lib/hadoop-streaming-2.7.6.jar
INPUT=/chap16/input/input.txt
OUTPUT=output
MAPPER=map.py
REDUCER=reduce.py
hadoop fs -rm -r  $OUTPUT
hadoop jar $JAR \
-D stream.map.input.ignoreKey=true \
-D stream.map.output.field.separator=, \
-D stream.num.map.output.key.fields=1 \
-D mapreduce.map.output.key.field.separator=, \
-D mapreduce.partition.keypartitioner.options='-k1,1' \
-D mapreduce.job.output.key.comparator.class=org.apache.hadoop.mapreduce.lib.partition.KeyFieldBasedComparator \
-D mapreduce.partition.keycomparator.options='-k1,1' \
-input $INPUT \
-output $OUTPUT \
-mapper $MAPPER \
-reducer $REDUCER \
-file $MAPPER \
-file $REDUCER \
-partitioner org.apache.hadoop.mapred.lib.KeyFieldBasedPartitioner
