naive Bayes classifier 朴素贝叶斯

Posted on 2019年8月3日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

朴素贝叶斯是一种线性分类器。应用之前，要假设属性之间是独立的。先要在训练数据集上得出分类器。然后用分类器分类新数据。

符号类的数据集方便使用贝叶斯分类器。

从训练数据中，可以得出哪些值的概率了？

易知道，有两大类：yes和no。可以求出P(yes)和P(no)。

在yes和no的两大类下，又可以确定这些属性的概率：*位置处为yes或者no。

P(cool | *),P(high | *) ,P(hot | *) ,P(mild | *) ,P(normal | *) ,P(overcast | *) ,P(rain | *) ,P(strong | *) ,P(sunny | *) ,P(weak | *)

举例说明如何计算这些值。

计算P(yes)。

统计yes类和no类的总个数。P(yes)就是yes类个数的百分比。

计算P(strong | no)。

统计strong,no组合的总个数。P(strong | no)就是这个组合个数占no类个数的百分比。

设计mapreduce任务训练分类器

基于上述的计算方法。总重要的就是组合字段的计数。

比如，要计算P(yes)，统计yes项的个数。要计算P(strong | no)，要统计no项的个数和strong,no项的个数。

map.py

将每行数据拆分成，发射yes和no项，以及与yes和no的组合项。

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split(',')
    cid = segs[-1]
    for x in segs[:-1]:
        print('%s,%s,1' % (x,cid))
    print('CLASS,%s,1' % (cid))

reduce.py

累加各项的计数。

#!/usr/bin/python
#coding=utf-8
import sys

last_key = ''
last_count = 0

def gen(key,count):
    '''
    '''
    print('%s\t%d' % (key,count))

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    key = segs[0]
    if last_key != key:
        if last_key != '':
            gen(last_key,last_count)
            last_key = ''
            last_count = 0
    last_key = key
    last_count += long(segs[1])

if last_key != '':
    gen(last_key,last_count)

有了计数结果，立即计算概率表。其逻辑如下：

#!/usr/bin/python
#coding=utf-8
import sys

class_dict = {}
key_dict = {}
p_dict = {}

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    key = segs[0]
    count = long(segs[1])
    if key.startswith('CLASS'):
        cid =  key.split(',')[1]
        if cid not in class_dict:
            class_dict[cid] = count
    else:
        if key not in key_dict:
            key_dict[key] = count

def gen_table():
    '''
    生成概率表
    '''
    #分类概率表
    class_count = 0
    for key,count in class_dict.items():
        class_count += count
    
    for key,count in class_dict.items():
        p_dict[key] = count / float(class_count)
    
    #条件概率表
    for key,count in key_dict.items():
        cid = key.split(',')[1]
        if cid not in class_dict:
            continue
        t_count = class_dict[cid]
        p_dict[key] = count / float(t_count)

    for key,p in p_dict.items():
        print('%s\t%f' % (key,p))

gen_table()

用分类器分类新数据

新数据：

rain,hot,high,strong
overcast,mild,normal,weak
sunny,mild,normal,weak

要判断每行的结果是yes还是no。

形式化表示朴素贝叶斯分类原理：

result C = arg max P(C=c | X)
         = arg max P(C=c,X) / P(X)
         = arg max P(X | C=c) * P(C=c) / P(x)
         = arg max P(X | C=c) * P(C=c)
         = arg max P(x1 | C=c)*...*P(xn | C=c) * P(C=c) (其中X=(x1,...,xn)，各个属性独立)

如此，分类实际上是各个属性在每个分类C下的条件概率与C的概率的乘积。取让概率最大的那个分类。

#!/usr/bin/python
#coding=utf-8
import sys

class_list = ['yes','no']
p_table_file = 'table.txt'
table_dict = {}

def load_p_table(table_file,table_dict):
    '''
    加载概率表
    '''
    with open(table_file) as tf:
        for line in tf:
            segs = line.split()
            #print(segs)
            table_dict[segs[0]] = float(segs[1]) 

load_p_table(p_table_file,table_dict)

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split(',')
    ans_cid = None
    ans_p = -1 
    for cid in class_list:
        p_c = table_dict[cid]
        for tid in segs:
            key = tid+','+cid
            if key in table_dict:
                p_c *= table_dict[key]
            else:
                p_c *= 0
        
        if p_c > ans_p:
            ans_cid = cid
            ans_p = p_c
    print('%s\t%s\t%f'%(line,ans_cid,ans_p))

#!/bin/bash
#hadoop 2.7.6
JAR=${HADOOP_HOME}/share/hadoop/tools/lib/hadoop-streaming-2.7.6.jar
INPUT=/chap14/input/input.txt
OUTPUT=output
MAPPER=map.py
REDUCER=reduce.py
REDUCE_OUTPUT=ro.txt
PROBILITY_TABLE=table.txt
hadoop fs -rm -r  $OUTPUT
hadoop jar $JAR \
-D stream.map.input.ignoreKey=true \
-D stream.map.output.field.separator=, \
-D stream.num.map.output.key.fields=2 \
-D mapreduce.map.output.key.field.separator=, \
-D mapreduce.partition.keypartitioner.options='-k1,2' \
-D mapreduce.job.output.key.comparator.class=org.apache.hadoop.mapreduce.lib.partition.KeyFieldBasedComparator \
-D mapreduce.partition.keycomparator.options='-k1,2' \
-input $INPUT \
-output $OUTPUT \
-mapper $MAPPER \
-reducer $REDUCER \
-file $MAPPER \
-file $REDUCER \
-partitioner org.apache.hadoop.mapred.lib.KeyFieldBasedPartitioner 

if [ -f "$REDUCE_OUTPUT" ]; then
    rm -f "$REDUCE_OUTPUT"
fi

#导出结果到文件
hadoop fs -getmerge ${OUTPUT}/* $REDUCE_OUTPUT

#生成概率表
cat $REDUCE_OUTPUT | python table.py > $PROBILITY_TABLE
