kmeans k均值聚类

Posted on 2019年8月1日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

kmeans原理不难理解。本文着重讲述kmeans的算法编写过程。

算法框架：

  1. 初始化质心；
  2. 执行mapreduce任务：为每个点选最近的质心。统计每个质心所含的点，并计算新质心；
  3. 判断新质心与旧质心之间的距离是否小于某个阈值。如果小于某个阈值，则算法结束；
  4. 判断迭代次数是否超过最大迭代次数。如果没有，则算法结束，否则再次执行1，2，3，4。

输入点数据：

1 2
1 3
1 4
2 5
2 6
2 7
2 8
3 100
3 101
3 102
3 103
3 104

初始质心：第一行表示质心的数量，就是k值。之后行表示各个质心。各个质心要不同。

3
1.0 2.0
1.0 3.0
1.0 4.0

map.py

加载质心。对每个点选择距离最近的质心。发射<质心，点>。

#!/usr/bin/python
#coding=utf-8
import sys

center_file = 'centers.txt'
centers = []

def load_centers(centers):
    '''
    加载质心
    '''
    #print('load_centers')
    with open(center_file) as cf:
        count_str = cf.readline()
        count = long(count_str)
        #print(count)
        for i in range(count):
            point_str = cf.readline()
            point_segs = point_str.split()
            centers.append((float(point_segs[0]),float(point_segs[1])))

def distance(x,y):
    '''
    计算x与y的距离
    '''
    diff_1 = x[0] - y[0]
    diff_2 = x[1] - y[1]
    return diff_1 * diff_1 + diff_2 * diff_2

load_centers(centers)

for line in sys.stdin:
    line = line.strip()
    segs = line.split()
    point = (float(segs[0]),float(segs[1]))
    min_dist = sys.float_info.max
    min_center = None 
    for center in centers:
        dist = distance(point,center)
        if min_center == None:
            min_dist = dist
            min_center = center
        else:
            if dist < min_dist:
                min_dist = dist
                min_center = center
        #print('%f,%f -- %f,%f = %f' % (point[0],point[1],center[0],center[1],dist))
    print('%s\t%s\t%s\t%s' % (min_center[0],min_center[1],point[0],point[1]))

reduce.py

聚合每个质心的所有点，算出新质心。发射<质心，新质心>。

#!/usr/bin/python
#coding=utf-8
import sys

last_center = ''
last_points = []

def gen(center,points):
    '''
    计算每个质心的新位置
    '''
    a = 0.0
    b = 0.0
    for x in points:
        a += x[0]
        b += x[1]
    print('%s\t%s\t%s\t%s' % (center[0],center[1],a/len(points),b/len(points)))

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    center = (float(segs[0]),float(segs[1]))
    point = (float(segs[2]),float(segs[3]))
    if center != last_center:
        if last_center != '':
            gen(last_center,last_points)
            last_center = ''
            last_points = []
    last_center = center
    last_points.append(point)

if last_center != '':
    gen(last_center,last_points)

判断终止：

两个终止条件，其中一个是判断质心变化是否小于某个阈值。这放在change.py中，同时在change.py中还要聚合更换新质心。

change.py

#!/usr/bin/python
#coding=utf-8
import sys

center_file = 'centers.txt'
new_center_file = 'new_centers.txt'
merge_file = "merge.txt"
centers = []
new_centers = []
threshold = float(0.05)

def load_centers(infile,centers):
    '''
    加载质心
    '''
    #print('load_centers')
    with open(infile) as cf:
        count_str = cf.readline()
        count = long(count_str)
        #print(count)
        for i in range(count):
            point_str = cf.readline()
            point_segs = point_str.split()
            centers.append((float(point_segs[0]),float(point_segs[1])))

def load_merge(infile,centers):
    '''
    加载mapreduce结果文件
    '''
    #print('load_centers')
    with open(infile) as cf:
        for point_str in cf:
            point_segs = point_str.split()
            centers.append((float(point_segs[2]),float(point_segs[3])))

def save_new_centers(new_center_file,centers):
    '''
    保存更新后的质心文件
    '''
    with open(new_center_file,"w") as ncf:
        ncf.write('%d\n' % (len(centers)))
        for x in centers:
            ncf.write('%s\t%s\n' % (x[0],x[1]))

load_centers(center_file,centers)
load_merge(merge_file,new_centers)
save_new_centers(new_center_file,new_centers)

def distance(x,y):
    '''
    计算x与y的距离
    '''
    diff_1 = x[0] - y[0]
    diff_2 = x[1] - y[1]
    return diff_1 * diff_1 + diff_2 * diff_2

def change(centers,new_centers):
    '''
    计算质心变化
    '''
    s = 0.0
    for old,new in zip(centers,new_centers):
        s += distance(old,new)
    print('centers changes from')
    print(centers)
    print('to')
    print(new_centers)
    return s < threshold

if change(centers,new_centers):
    sys.exit(0)
else:
    sys.exit(-1)

最大迭代次数的终止条件放在提交脚本里。

提交脚本：

#!/bin/bash
#hadoop 2.7.6
JAR=${HADOOP_HOME}/share/hadoop/tools/lib/hadoop-streaming-2.7.6.jar
INPUT=/chap12/input/input.txt
OUTPUT=output
MAPPER=map.py
REDUCER=reduce.py
CENTERS=centers.txt
NEW_CENTERS=new_centers.txt
MERGE_FILE=merge.txt
MAX_ITERATION=10

do_mapreduce_job()
{
hadoop fs -rm -r  $OUTPUT
hadoop jar $JAR \
-D stream.map.input.ignoreKey=true \
-D stream.map.output.field.separator='\t' \
-D stream.num.map.output.key.fields=2 \
-D mapreduce.job.output.key.comparator.class=org.apache.hadoop.mapreduce.lib.partition.KeyFieldBasedComparator \
-D mapreduce.partition.keycomparator.options=-k1,2 \
-input $INPUT \
-output $OUTPUT \
-mapper $MAPPER \
-reducer $REDUCER \
-file $MAPPER \
-file $REDUCER \
-file $CENTERS \
-partitioner org.apache.hadoop.mapred.lib.KeyFieldBasedPartitioner 
if [ -f "$MERGE_FILE" ]; then
    rm -f "$MERGE_FILE"
fi
#导出结果到新文件
hadoop fs -getmerge output/* $MERGE_FILE
}

kmeans()
{
counter=1
while [ "$counter" -le "$MAX_ITERATION" ]
do
    echo "The $counter th iteration"
    #做mapreduce任务
    do_mapreduce_job
    #判断是否要继续执行迭代过程
    python change.py
    if [ $? -eq 0 ]; then
        echo "mapreduce succeed"
        echo "has done $counter iterations"
        echo "final centers are:"
        cat $NEW_CENTERS
        break
    fi
    #更新质心文件
    cat $NEW_CENTERS > $CENTERS
    counter=$(( $counter + 1))
done

if [ "$counter" -eq "$MAX_ITERATION" ];then
    echo "mapreduce exceed max iteration"
    echo "has done $counter iterations"
    echo "final centers are:"
    cat $NEW_CENTERS
fi
}
