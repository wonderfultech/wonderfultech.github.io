leetcode215 数组中的第K个最大元素 Kth Largest Element in an Array

Posted on 2019年4月12日2019年12月4日by jasonpeng

leetcode 数据库题目全部题解

Table of Contents

  • 解法一 小顶堆
  • 解法二 快排划分法

解法一 小顶堆

思路: 小顶堆中始终保存k个最大元素. 从左向右遍历数组, 将A[i]加入堆, 如果堆中元素个数超过k个, 出堆.

例如: [3,2,1,5,6,4] 和 k=2

3 入堆 . 堆=[3]

2 入堆 . 堆=[3,2]

1 入堆 . 堆=[3,2,1] 超过2个. 1出堆, 堆=[3,2]

5入堆, 堆=[5,3,2] 超过2个. 2出堆, 堆=[5,3]

6入堆, 堆=[6,5,3] 超过2个. 3出堆, 堆=[6,5]

4入堆, 堆=[6,5,4] 超过2个. 4出堆, 堆=[6,5]

结果是5

完整思路:

Q;堆
for(i < N){
    A[i]入Q;
    if(Q.length > k) Q.top出堆;
}
返回Q.top;

代码:

    int findKthLargest(vector<int>& nums, int k) {
        priority_queue<int,vector<int>,greater<int>> Q;
        for(int i=0;i<nums.size();++i){
            Q.push(nums[i]);
            if(Q.size() > k) Q.pop();
        }
        return Q.top();
    }

解法二 快排划分法

借鉴快速排序的划分思想, 先将数组分成三个部分:[ >枢轴元素; =枢轴元素 ; <枢轴元素].

划分后, 枢轴元素的最终位置为p.

当 k == p 时, A[p]就是结果;

当 k > p 时, 继续在A[p+1,...,N-1]中寻找;

当 k < p 时, 继续在A[0,...,p-1]中寻找;

例如:[3,2,1,5,6,4] 和 k=2

枢轴元素选4.

3<4, 交换3和6, [6,2,1,5,3,4];

6>4, 位置不变,下一个元素; [6,2,1,5,3,4];

2<4, 交换2和5, [6,5,1,2,3,4];

5>4 , 位置不变,下一个元素; [6,5,1,2,3,4]

1<4,1与1交换; [6,5,1,2,3,4]

枢轴元素4的最终位置是1的位置; [6,5,4,1,2,3] .

显然, 接下来要在4左边递归.

思路:

//[x,y)左闭右开区间
kth(A,x,y,k){
    if(x >= y) 返回-1;
    if(x+1 == y) 一个元素,返回A[x];
    //枢轴元素
    p=A[y-1];
    //[x,more) > 枢轴元素
    more=x;
    //(less,y) < 枢轴元素
    //(more,i) =枢轴元素 
    i = x;
    while(i <= less){
            if(A[i] > p){
                //A[i]放入[x,more)
                交换A[i++],A[more++];
            }else if(A[i] < p){
                //A[i]放入(less,y)
                交换A[i],A[less--];
            }else ++i; //(more,i)
    }
    //确定最终枢轴元素位置为less+1
    交换A[less+1],A[y-1];
    if(less+1 == k) 返回A[k];
    else if(less+1 > k) A[x,...,less+1)中寻找,递归;
    else A[less+2,...,y)中寻找,递归;
}

代码:

    int findKthLargest(vector<int>& nums, int k) {
        return kth(nums,0,nums.size(),k-1);
    }
    
    //[x,y)
    int kth(vector<int>& A,int x,int y,int k){
        if(x >= y) return -1;
        if(x+1 == y) return A[x];
        const int& p=A[y-1];
        int more=x,less=y-2;
        int i=x;
        while(i <= less){
            if(A[i] > p){
                swap(A[i++],A[more++]);
            }else if(A[i] < p){
                swap(A[i],A[less--]);
            }else ++i;
        }
        swap(A[less+1],A[y-1]);
        if(less+1 == k){
            return A[k];
        }else if(less+1 > k){
            return kth(A,x,less+1,k);
        }else return kth(A,less+2,y,k);
    }
