解法一 牛顿法

相较于二分法, 本题用牛顿法求解, 写码时边界条件不易出错. 牛顿老师诚不欺我!

其难点在于公式的推导.

关键步骤在下图中.保证牛顿法正确求解的条件时,

一阶导数:f'(a)=2a >=0 ,表示f(a)单调递增.

二阶导数:f''(a)=2 >0 ,表示f(a)在[0,x]上是下凸函数.这样每一点的切线一定在曲线下方.

本题求x的平方根,等价于在[0,x]上求方程(1), f(a) = 0 的整数解.

求过点(a,f(a))的切线与x轴的交点(b,0).

公式(2)是切线斜率.

公式(3)是切线方程.

在 公式(3) 中令y=0, 得出方程(4).

再将方程(1)代入方程(4)得出方程(5), 整理得出方程(6)(7).

再一次算,过点(b,f(b))的切线与x轴的交点, 即公式(8),重复上面的计算过程.

sqrt(x){
    a=x/2;
    while(a*a > x){
        a = (a+x/a)/2;
    }
    返回a;
}

代码:

    typedef long long ll;
    int mySqrt(int x) {
        if(x == 1) return 1;
        ll A=x/2;
        while(A*A > x){
            A = (A + x/A)/2;
        }
        return A;
    }

解法二 二分法

从理解角度, 二分法比牛顿法容易理解, 但是编码容易出现错误.

一次性写出个正确的解法并不容易.

二分法思路:在区间[a,b]取中间值m,判断m*m与x的大小关系.

当 m*m > x 时 表示, m大了. 那么接下来在[1,m-1]取中间值再判断.

当 m*m < x时 表示, m小了. 那么接下来在[m+1,x]取中间值再判断.

sqrt(x){
    a=1,b=x+1;
    while(a < b){
        m=(a+b)/2;
        if(m <= x/m) a=m+1;
        else b=m;
    }
    返回a-1;
}

有四种类似的实现:

第一种:

class Solution {
public:
    int mySqrt(int x) {
        long long a=1,b=(long long)x+1;
        while(a<b){
            long long m=a+(b-a)/2;
            if(m<=x/m) a=m+1;
            else b=m;
        }
        return a-1;
    }
};

第二种:

class Solution {
public:
    int mySqrt(int x) {
        if(!x) return x;
        int a=1,b=x;
        while(a<b){
            int m=a+(b-a)/2;
            int d=x/m;
            if(m<d) a=m+1;
            else b=m;
        }
        if(a>(x/a)) return a-1;
        return a;
    }
};

第三种:

class Solution {
public:
    int mySqrt(int x) {
        if(!x) return x;
        int a=1,b=x;
        while(a<=b){
            int m=a+(b-a)/2;
            int d=x/m;
            if(d==m) return m;
            if(m<d) a=m+1;
            else b=m-1;
        }
        return b;
    }
};

第四种:

class Solution {

    }
};
