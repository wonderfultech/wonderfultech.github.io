leetcode903 DI 序列的有效排列 Valid Permutations for DI Sequence

解法一 回溯法（超时）

本题的回溯解法，类似 leetcode47 全排列 II，leetcode46 全排列的解法。

问题转变为有约束的全排列问题。即，在位置i，放置{0,1,...,n}中的哪个数字。既不重复使用数字，也能满足条件。直到n+1个位置都放满数字。

思路，在全排列的回溯算法框架中，加上减栈。筛选正确的排列，并计数。

全排列的回溯算法。 参考 leetcode47 全排列 II，leetcode46 全排列的讲解。这里直接给出算法。

其中，减栈条件：

当 S[cur-1]=='D' 且 i >path[cur-1]时，减栈;

当 S[cur-1]=='I' 且 i <path[cur-1]时，减栈;

S;题中的输入串
dfs(S,path,cur){
    if(cur == S.length+1){
        一个合法排列，计数;
    }else for(i in [0,...,S.length]){
        if(cur){
            if(S[cur-1] == 'D' 且 i > path[cur-1]) pass;
            if(S[cur-1] == 'I' 且 i < path[cur-1]) pass;
        }

        if(i not in path[0,...,cur-1]){
            path[cur] = i;
            dfs(S,path,cur+1);
        }
    }
}

代码（超时）：

const int mode=1000000007;
class Solution {
public:
    int ans;
    int numPermsDISequence(string S) {
        ans = 0;
        vector<int> path(S.size()+1);
        dfs(S,path,0);
        
        return ans;
    }
    
    void dfs(const string& S,
            vector<int>& path,
             int cur){
        if(cur == path.size()){
            ans=(ans%mode+1)%mode;
        }else for(int i=0;i<=S.size();++i){
            if(cur){
                if(S[cur-1]=='D' && i > path[cur-1]){
                    continue;
                }else if(S[cur-1] =='I' && i < path[cur-1]){
                    continue;
                }
            }
            bool ok=true;
            for(int j=0;j<cur;++j){
                if(path[j] == i){
                    ok=false;
                    break;
                }
            }

            if(ok){
                path[cur] = i;
                dfs(S,path,cur+1);
            }
        }
    }
};

由于全排列的算法复杂度是O(n!)。上面的算法在n<20以内可以通过，超过n>=20就会超时。

解法二 回溯法

在解法一的基础上，进一步挖掘。发现有很多重复计算。要避免掉这些重复计算。 借鉴 518. 零钱兑换 II的回溯解法 。

重复计算在什么地方？

在位置i放置好后数字后，回溯后，再一次来到位置i，是否可以避免重复计算？

字符串S，控制着位置i处的数字与位置i-1处数字的大小关系。

单独一个位置i不能确定放置情况，还要明确位置i处的数字。

设d[i][j]表示在位置i放置数字j时的排列数。

初始d[i][j]=-1，表示没有计算过。

当d[i][j] > -1时，表示d[i][j]已经算过。不用再重复计算。

在解法一上，将d[i][j]去重加入进去。

同时，要修改终止条件，当串S遍历完后，计数为1.

S;题中的输入串
d[i][j]=-1;
dfs(S,path,cur,d){
    ans = 0;//总计数
    if(cur == S.length+1){
        返回1;
    }else for(i in [0,...,S.length]){
        if(cur){
            if(S[cur-1] == 'D' 且 i > path[cur-1]) pass;
            if(S[cur-1] == 'I' 且 i < path[cur-1]) pass;
        }

        if(i not in path[0,...,cur-1]){
            path[cur] = i;
            if(d[cur][i] == -1){
                d[cur][i] = dfs(S,path,cur+1,d);
            }
            ans += d[cur][i];
        }
    }
    返回ans;
}

代码：

const int mode=1000000007;
class Solution {
public:
    int ans;
    int numPermsDISequence(string S) {
        ans = 0;
        vector<int> path(S.size()+1);
        vector<vector<int>> d(S.size()+1,vector<int>(S.size()+1,-1));
        dfs(S,path,0,d);
        for(int i=0;i<=S.size();++i){
            if(d[0][i] > -1){
                ans=(ans%mode + d[0][i]%mode)%mode;
            }
        }
        return ans;
    }
    
    int dfs(const string& S,
            vector<int>& path,
             int cur,
             vector<vector<int>>& d
            ){
        int ans=0;
        if(cur == path.size()){
            return 1;
        }else for(int i=0;i<=S.size();++i){
            if(cur){
                if(S[cur-1]=='D' && i > path[cur-1]){
                    continue;
                }else if(S[cur-1] =='I' && i < path[cur-1]){
                    continue;
                }
            }
            bool ok=true;
            for(int j=0;j<cur;++j){
                if(path[j] == i){
                    ok=false;
                    break;
                }
            }

            if(ok){
                path[cur] = i;
                int& b=d[cur][i];
                if(b == -1){
                    b=0;
                    b = dfs(S,path,cur+1,d);
                }
                
                ans =(ans%mode + b%mode)%mode;
            }
        }
        return ans;
    }
};

解法三 动态规划

将解法二递归转换成递推。

重复数字问题

思路不难想到，但是被一个问题卡住了。

就是数字去重复问题。

例如：S="DI"

排列是:[1,0,2]和[2,0,1]。

现在，S尾部增加一个"D"。新增数字3。

现在问题来了，这个数字3怎么加入了？

数字3放在尾部，肯定是不对的。因为S新增的D，要求数字下降。但是数字3是最大值。

[1,0,2]的尾部，只能放0和1.

可0已经出现过了。解决方法是，将前面的0改为1。 但是1又会重复。再将前面的1改为2. 2又会重复，再将前面的2改为3. 最后变成[2,1,3]，再在尾部添加0，即可。既没重复，又用到了数字3.

综合看，串S新增D时，尾部数字j，尾部只能增加[0,...,x,...,j-1]中的数字。当要增加数字x时，序列中所有 大于等于 x的数字都加1，可避免数字x的重复。

同样，S尾部增加一个"I"。新增数字3。

序列[1,0,2]可以直接放3，也只能放3. 变成序列[1,0,2,3] 。而不违反"I"的要求。

序列[2,0,1]之后可以放2和3. 放3同上面的情况一样。

放2，数字2重复。采用与上面相同的思路，将序列中所有 大于等于 x的数字都加1.

综合看，串S新增I时，尾部数字j，尾部只能增加[j+1,...,x,...,N]中的数字。当要增加数字x时，序列中所有 大于等于 x的数字都加1，可避免数字x的重复。

递推

设S(i)表示S的长度为i的前缀串。i==0时，表示空串。S[i-1]是前缀的最后一个字符。

注意S(i)能容许的最大数字是i，而不是N。

设d[i][j]表示当S(i)时，位置i放置数字j时的排列数。 初始d[i][j]=0。

当S[i-1] == 'D'时，d[i][j]受d[i-1][k]的影响。k在[j,...,i-1]中。

k能选j，因为此时可通过上面的方法去重数字j。

k不能选i，因为i是最大数字。

当S[i-1] == 'I'时，d[i][j]受d[i-1][k]的影响。k在[0,...,j-1]中。

k不能选j，因为如果要求前面的数字减1的话，可能会出现-1，越界的情况。

d[i][j]=0;
for(i in [0,S.length]){
    if(i==0){
        d[i][...]=1;
    }else{
        for(j in [0,i]){
            if(S[i-1]=='D'){
                for(k in [j,i-1]){
                    d[i][j] += d[i-1][k];
                }
            }else{
                for(k in [0,j-1]){
                    d[i][j] += d[i-1][k];
                }
            }
        }
    }
}
sum(d[S.length][...])是结果;

代码：

const int mode=1000000007;
class Solution {
public:
    int numPermsDISequence(string S) {
        vector<vector<int>> d(S.size()+1,vector<int>(S.size()+1,0));
        for(int i=0;i<=S.size();++i){
            if(i == 0){
                for(int j=0;j<=S.size();++j){
                    d[i][j] = 1;
                }
            }else{
                for(int j=0;j<=i;++j){
                    if(S[i-1] == 'D'){
                        for(int k=j;k<i;++k){
                            d[i][j] = (d[i][j] % mode + d[i-1][k] % mode) % mode;
                        }
                    }else{
                        for(int k=0;k<j;++k){
                            d[i][j] = (d[i][j] % mode + d[i-1][k] % mode) % mode;
                        }
                    }
                }
            }
        }
        
        int ans=0;
        for(int j=0;j<=S.size();++j){
            ans=(ans % mode + d[S.size()][j] % mode) % mode;
        }
        
        return ans;
    }
};
