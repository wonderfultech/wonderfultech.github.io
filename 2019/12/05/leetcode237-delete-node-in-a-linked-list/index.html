leetcode237 删除链表中的节点 Delete Node in a Linked List

Posted on 2019年12月5日2019年12月5日by jasonpeng

leetcode 数据库题目全部题解

解法一 交换法

本题要用好条件“ 链表中所有节点的值都是唯一的。 ”

那么删掉一个节点，就只需将其值抹去。并且最终链表要少个节点。

问题是，少掉的节点一定不能是要删去的节点。

因为，题目只给了要删除的节点的指针。其前面的节点指针一无所知。

但是可以删去后面的节点。

因此，将后面节点的值复制到要删除的节点，并将后面节点删掉。

node.val=node.next.val;
node.next=node.next.next;

代码：

    void deleteNode(ListNode* node) {
        node->val = node->next->val;
        node->next = node->next->next;
    }
