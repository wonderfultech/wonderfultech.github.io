leetcode392 判断子序列 Is Subsequence

解法一 顺序查找法

设字符s[i] 与 t[j]相同。

那么，对s[i+1]来说，要从t[j+1]的位置开始继续查找。

例如： s = "abc", t = "ahbgdc"

s[0]='a' 与 t[0]='a'相同，那么，s[1]在t[1]位置开始继续查找。

i=0;
j=0;
while(i < s.length){
    while(j < t.length 且 s[i] != t[j]) ++j;
    if(j >= t.length) 返回false;
    ++j;++i;
}
此时，true.

代码：

    bool isSubsequence(string s, string t) {
        int i=0;
        int j=0;
        while(i < s.size()){
            while(j < t.size() && t[j] != s[i]) ++j;
            if(j >= t.size()) return false;
            ++j;
            ++i;
        }
        return true;
    }

解法二 动态规划

本题实际上是变形的正则表达式。

可以参考10. 正则表达式匹配 的解法

设s(i)表示长度为i的前缀串。t(i)同理。

s(i)最后一个元素是s[i-1]。同理t[j-1]。

设d[i][j]表示s(i)与t(j)的匹配情况。

表格d[i][j]表示s(i)和t(j)的匹配情况。

分情况讨论。

当s(0)，与t(0)时，两个空串。匹配成功。d[0][0]=1。

当s(0),t(j),j>0时，此时s(0)空串，t(j)非空。

d[0][j]=1。

当s(i),i>0,t(0)时，此时t(0)空串，s(i)非空。匹配失败。

d[i][0]=0。

当s(i),i>0,t(j),j>0时，分情况讨论。

当s[i-1] == t[j-1]时，s(i)与t(j)的匹配情况，取决于s(i-1)于t(j-1)的匹配情况。

d[i][j]=d[i-1][j-1]。

例如：s = "axc" t= "adc"

s[2]=c 与 t[2]=c 相同，但是s(2)与t(2)不匹配。

当s[i-1] != t[j-1]时，s(i)与t(j)的匹配情况，取决于s(i)于t(j-1)的匹配情况。

d[i][j]=d[i][j-1]

例如： s = "axc" t= "axdc"

s[1]=x 与 t[2]=d，不相同。但是s(2)与t(2)匹配，因此s(2)与t(3)也匹配。

    bool isSubsequence(string s, string t) {
        vector<vector<int>> d(s.size()+1,vector<int>(t.size()+1,0));
        for(int i=0;i<=s.size();++i){
            for(int j=0;j<=t.size();++j){
                if(i == 0 && j == 0){
                    d[i][j] = 1;
                }else if(i == 0 && j > 0){
                    d[i][j] = 1;
                }else if(i > 0 && j == 0){
                    d[i][j] = 0;
                }else{
                    if(s[i-1] == t[j-1]){
                        d[i][j] = d[i-1][j-1] && 1;
                    }else{
                        d[i][j] = d[i][j-1];
                    }
                }
            }
        }
        
        return d[s.size()][t.size()];
    }

降到二维滚动数组

但是直接用上面的解法提交，会超时。会碰到一个t很长的case。

从上面的解法中，可看出d[i][j]只与d[i-1][j-1]和d[i][j-1]有关，因此d只需要两行即可。

用二维滚动数组代替d。

    bool isSubsequence(string s, string t) {
        vector<vector<int>> d(2,vector<int>(t.size()+1,0));
        for(int i=0;i<=s.size();++i){
            for(int j=0;j<=t.size();++j){
                if(i == 0 && j == 0){
                    d[i&1][j] = 1;
                }else if(i == 0 && j > 0){
                    d[i&1][j] = 1;
                }else if(i > 0 && j == 0){
                    d[i&1][j] = 0;
                }else{
                    if(s[i-1] == t[j-1]){
                        d[i&1][j] = d[(i-1)&1][j-1] && 1;
                    }else{
                        d[i&1][j] = d[i&1][j-1];
                    }
                }
            }
        }
        
        return d[s.size()&1][t.size()];
    }

解法三 二分查找法

对解法一进行优化。

设字符s[i] 与 t[j]相同。

那么，对s[i+1]来说，要从t[j+1]的位置开始继续查找。

对s[i+1]的查找，不可避免的会碰到其它字符，多出不必要的运算。

而实际上，只需要找一个下标比i大的s[i+1]。

建立一个哈希表hash<字符，下标序列>。

对字符串t预先处理。将每个字符t[j]的下标j存起来。而且下标序列，一定是递增排列。

hash<字符,[]>;
for(j < t.length){
    j入hash[t[j]];
}

当s[i] == t[j]时，

对每个s[i+1]，只需从hash[s[i+1]]中找一个比j大下标。用二分查找upper_bound即可解决。

upper_bound(A,x)找出有序序列A中，比x大的第一个元素。

j=-1;
for(i < s.length){
    k=upper_bound(hash[s[i]],j);
    if(k没找到) 返回失败;
    j=k;
}

代码：

    bool isSubsequence(string s, string t) {
        vector<vector<int>> hash(26,vector<int>());
        for(int i=0;i<t.size();++i){
            hash[t[i]-'a'].push_back(i);
        }
        
        int pre=-1;
        for(int i=0;i<s.size();++i){
            vector<int>& choices=hash[s[i]-'a'];
            vector<int>::iterator j = upper_bound(choices.begin(),choices.end(),pre);
            if(j == choices.end()) return false;
            pre=*j;
        }
        
        return true;
    }
