leetcode1183 矩阵中 1 的最大数量 Maximum Number of Ones

Posted on 2019年12月29日2019年12月29日by jasonpeng

leetcode 数据库题目全部题解

Table of Contents

  • 解法一 回溯算法（超时）
  • 解法二 贪心算法

解法一 回溯算法（超时）

尽管提示中说明要贪心算法。但一开始并没有想到用贪心算法怎么求解。

遂走了老方法，用回溯法求解。

回溯法的思路：在任何一个位置(x,y)，检测所有包含它的正方形，只要有一个正方形中1的个数超过了限制，说明1不能放在该处。否则，1就可以放在此处。

当然，如果(x,y)确实可以放置1，也有两种选择。跳过不放置1，或放1.

如此，直到最后一个位置为止。

当考虑完全部位置后，再统计1的个数。

当然，在题目中的输入数据量下，效率感人。

利用提示2，将回溯法简化。

仅考虑将1放置在第一个正方形。

放置1时，利用提示2，将全部要放置的1都计算出来。

ans=0;//结果
dfs(x,y,cur,cnt){
    if(cur==maxOnes 或 x >= sideLength 或 y >= sideLength){
         ans=max(ans,cnt);
         返回;
    }
    //在(x,y)处放置1
    利用提示2计算要放置的全部1的个数t;
    if(y+1<L){
        dfs(x,y+1,cur+1,cnt+t);
    }else{
        dfs(x+1,0,cur+1,cnt+t);
    }
    if(y+1<L){
        dfs(x,y+1,cur,cnt);
    }else{
        dfs(x+1,0,cur,cnt);
    }
}

代码：

    int R,C,L,MAX;
    int ans;
    int maximumNumberOfOnes(int width, int height, int sideLength, int maxOnes) {
        R=height;
        C=width;
        L=sideLength;
        MAX=maxOnes;
        ans=-1;
        
        dfs(0,0,0,0);
        return ans;
    }

    void dfs(int x,int y,int cur,int cnt){
        if(cur == MAX){
            ans=max(ans,cnt);
            return ;
        }
        if(x>=L || y>=L) {
            ans=max(ans,cnt);
            return ;
        }
        int t=set(x,y);
        if(y+1<L){
            dfs(x,y+1,cur+1,cnt+t);
        }else{
            dfs(x+1,0,cur+1,cnt+t);
        }
        if(y+1<L){
            dfs(x,y+1,cur,cnt);
        }else{
            dfs(x+1,0,cur,cnt);
        }
        return ;
    }
    
    int set(int x,int y){
        int i,j;
        int cnt=0;
        for(int k=0;(i=(k*L+(x % L))) < R;++k){
            for(int t=0;(j=(t*L+(y % L))) < C;++t){
                ++cnt;
            }
        }
        return cnt;
    }

解法二 贪心算法

在解法一中，在第一个正方形中放置1。只要正方形中1的个数没有超过1. 位置之间，没什么关系。

在 第一个正方形中 不同位置上放置1，每个位置能得出1的个数，是不一样的。

问题就简化为 在 第一个正方形中 哪些位置上放置1 ，既能不超过限制，又能使1的总个数最大。

容易想到，第一个正方形中1的个数应该用到最多，即maxOnes。

实际上，就变成了。在 第一个正方形中 挑出maxOnes个位置。这些位置上，产生1的个数是最大的。

ans=[];
for(x < L){
    for(y < L){
        位置(x,y)处放置1时，能生成的1的个数t;
        t入ans;
    }
}
ans中前maxOnes个最大值之和既是结果;

代码：

    int R,C,L,MAX;
    int ans;
    int maximumNumberOfOnes(int width, int height, int sideLength, int maxOnes) {
        R=height;
        C=width;
        L=sideLength;
        MAX=maxOnes;
        ans=0;

        priority_queue<int,vector<int>,greater<int>> Q;
        for(int x=0;x<L;++x){
            for(int y=0;y<L;++y){
                int r=setnone(x,y,1);
                Q.push(r);
                if(Q.size() > MAX){
                    Q.pop();
                }
            }
        }
        
        while(Q.size()){
            ans += Q.top();Q.pop();
        }
        return ans;
    }
    
    int setnone(int x,int y,int v){
        int i,j;
        int cnt=0;
        for(int k=0;(i=(k*L+(x % L))) < R;++k){
            for(int t=0;(j=(t*L+(y % L))) < C;++t){
                ++cnt;
            }
        }
        return cnt;
    }
