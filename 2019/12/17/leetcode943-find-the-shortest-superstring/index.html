leetcode943 最短超级串 Find the Shortest Superstring

解法一 动态规划

本题是题目 UVA1204的简化版。尽管是简化版，但也不容易。

最简单的思路是算出A中字符串的全排列。针对每个排列，去除相邻字符串间的重叠部分。求出最短长度即可。

显然这种方法时间效率不高。但是可以分析出一个点，字符串间的顺序很关键。

设集合S是A的子集。给定集合S，并不能确定最短超级串，还要确定s中字符串的顺序。

从S是空集考虑。

当S是空集时，新插入A中的任意一个字符串a。之后，S={a}，那么S的最短超级串为a。

当S不是空集时， 新插入A中的任意一个不在S中的字符串a。即a in A-S.

a插入之前，S中的字符串顺序不确定。

a插入之后，S中的字符串也不确定。

这样的复杂性，难以确定S+a的最短超级串。

集合定序

难点，解决S中元素顺序不确定性。x插入顺序的不确定性。

对集合S进行限制。

集合S, 以S中字符串x为结尾的最短超级串T(S,x)。

集合S中的字符串可以组成很多超级串，这些超级串中以x为结尾的最短超级串用T(S,x)表示。

限定x是最后一个插入S的字符串。在集合S-{x}中，插入x，就以x为结尾形成最短超级串T(S,x)。

基于此，对S中的每个字符串y，将y选为最后插入的字符串。

形成转移，在S-{y}中，插入y，形成最短超级串T(S,y).

对集合S-{y}做同样的考虑，递归下去，直到S为空。这就确定了一系列集合，对应形成一系列最短超级串，它们的尾部字符串都是确定的。

同样，向集合S插入字符串z，能形成转移：S到S+z，T(S+z,z)。

从S可以发生的转移：

for(y in S){
    for(x in A){
        if(x in S) pass掉;
        S -> S+x;
        T(S,y) -> T(S+x,x);
    }
}

集合表示

用位图的形式表示集合S，恰好用一个2进制整数表示集合S。设A中元素个为N。

结合位运算，就可以对集合S操作进行编码。

1，空集，S=0. 全集，S=(1 << N) - 1.

2， 判断A[i]是否在S中，

A[i]在S中，即S中的第i为1。 A[i]不在S中，即S中的第i为0。

S & (1 << i) == 1 在S中, S & (1 << i) == 0 不在S中。

3, 向集合S中添加元素j，

S = S | (1 << j)

字符串重叠

字符串x与字符串y，可能存在重叠关系。题目中已经排除了x与y的包含关系。

x之间顺序也决定了，重叠部分可能不一样。

例如：x="abc",y="cdab"

x在前，y在后，重叠部分为c。

x比y短，

从x[1]与y[0]开始比较。

x[1] =a 不等于 y[0]=c

从x[2]与y[0]开始比较。

x[2] =c 等于 y[0]=c

y在前，x在后，重叠部分为ab。

y比x长，

从y[1]与x[0]开始比较。

y[1]=d 不等于 x[0]=a

从y[2]与x[0]开始比较。

y[2]=a 等于 x[0]=a

y[3]=b 等于 x[1]=b

计算x在前，y在后，最大重叠部分长度的思路：

overlap(x,y){
    p=max(0,x.length-y.length)+1;
    for(p < x.length){
        if(strncmp(x[p:],y,x.length-p)==0){
            最大重叠部分长度为x.length-p，返回
        }
    }
}

在本题中，数组中元素A[i],A[j]之间的最大重叠部分长度，可以事先计算出来。

设B[i][j]表示A[i]与A[j] 之间的最大重叠部分长度

for(i < A){
    for(j < A){
        if(i==j) B[i][j]=A[i].length;
        else B[i][j]=overlap(A[i],A[j]);
    }
}

递推算法

设d[S][i]表示集合S，以S中字符串A[i]为结尾时，重叠部分长度总和。

初始S=0. d[0][i]=1.

目标求d[S=(1<<N)-1][0,...,N-1]最大值。重叠部分长度总和越长，结果的超级串越短。

向集合S中，插入不在S中的字符串A[j]，并将A[j]作为S | (1<<j)的最后一个字符串.

从S转移到S | (1 << j)，并且数值大小上有，S < (S | (1<<j))。

那么，这就意味着，从S=0，往S=(1<<N)-1枚举时，S | (1<<j)不会小于等于S。保持偏序关系。保证d[S][i]是最终值。保证递推不会出错。

递推关系:

d[S | (1 << j)][j] = max( d[S | (1 << j)][j] ,d[S][i]+B[i][j]);

for(s in [0,1,2,...,(1<<N)-1]){
    for(i in A){
        if(i not in s) continue;
        for(j in A){
            if(j in s) continue;
            d[s|(1<<j)][j] = max(d[s|(1<<j)][j],d[s][i]+B[i][j]);
        }
    }
}

为了便于最后重建最短超级串。

设prev[s][i]表示形成d[s][i]的上一个d[prev_s][prev_i]。即 d[s][i] = d[prev_s][prev_i] + B[prev_i][i]。

将上面的递归关系加入prev.

for(s in [0,1,2,...,(1<<N)-1]){
    for(i in A){
        if(i not in s) continue;
        for(j in A){
            if(j in s) continue;
            if(d[s|(1<<j)][j] <= d[s][i]+B[i][j]){
                d[s|(1<<j)][j] = d[s][i]+B[i][j];
                prev[s|(1<<j)][j] = {s,i};
            }
        }
    }
}

重建 最短 超级串

基于d[(1<<N)-1][i]和 prev[(1<<N)-1][i] 进行构建。

计算 d[(1<<N)-1][0,...,N-1] 的最大值。

设最大值为 d[(1<<N)-1][maxi] ，这表示A中全部元素都用上后，重叠部分长度的最大值。

当 d[(1<<N)-1][maxi] == 0时，没有重叠部分。最短超级串就是A中字符串的拼接结果。

当 d[(1<<N)-1][maxi] > 0时，重叠部分长度最长。

从 [(1<<N)-1][maxi] 借助prev [(1<<N)-1][maxi] ,往前找出前一个字符串。直到集合为空。将找出的字符串序号放入数组V。并且这样找出的字符串顺序刚好与正确的拼接顺序相反。构造时要注意。

maxi=0;
for(i < A){
    if(d[(1<<N)-1][i] > d[(1<<N)-1][maxi]){
        maxi=i;
    }
}
if(d[(1<<N)-1][maxi] == 0) 拼接A中字符串，返回;
V=[];字符串索引
s=(1<<N)-1;
while(s){
    maxi入v头部;
    {prev_s,prev_i}=prev[s][maxi];
    s=prev_s;
    maxi=prev_i;
}

逆序遍历V，结合B[i][j]数组，构造最短超级串。

ans="";结果
for(i in reverse(V)){
    if(i == 0) ans += A[V[i]];
    else{
        //扣除重叠部分
        overlap_len = B[V[i-1]][V[i]];
        ans += A[V[i]][overlap_len:];
    }
}

完整代码

    string shortestSuperstring(vector<string>& A) {
        vector<vector<int>> B(A.size(),vector<int>(A.size(),0));
        for(int i=0;i<A.size();++i){
            for(int j=0;j<A.size();++j){
                if(i==j) B[i][j] = A[i].size();
                else B[i][j] = overlap(A[i],A[j]);
            }
        }
        int S=(1<<A.size());
        vector<vector<int>> d(S,vector<int>(A.size(),0));
        vector<vector<pair<int,int>>> prev(S,vector<pair<int,int>>(A.size()));
        for(int s=0;s<S;++s){         
            for(int i=0;i<A.size();++i){
                if(!(s & (1<<i))) continue;
                for(int j=0;j<A.size();++j){
                    if(s & (1 << j)) continue;
                    int& ans=d[s|(1<<j)][j];
                    int r=d[s][i] + B[i][j];
                    if(ans <= r){
                        ans = r;
                        prev[s|(1<<j)][j] = {s,i};
                    }
                }
            }
        }
        
        int maxi=0;
        for(int i=0;i<A.size();++i){
            if(d[S-1][i] > d[S-1][maxi]){
                maxi = i;
            }
        }
        
        if(d[S-1][maxi] == 0){
            string ans;
            for(auto x:A){
                ans+=x;
            }
            return ans;
        }
        
        int s=S-1;
        vector<int> v;
        while(s){
            v.push_back(maxi);
            pair<int,int> p=prev[s][maxi];
            s = p.first;
            maxi=p.second;
        }
        string ans;
        for(int i=v.size()-1;i>=0;--i){
            if(i == v.size()-1){
                ans += A[v[i]];
            }else{
                int over = B[v[i+1]][v[i]];
                ans += A[v[i]].substr(over);
            }
        }
        
        return ans;
    }
    
    int overlap(const string& a,const string& b){
        int p=max(0,(int)(a.size()-b.size()))+1;
        for(int i=p;i<a.size();++i){
            if(strncmp(a.c_str()+i,b.c_str(),a.size()-i) == 0){
                return a.size()-i;
            }
        }
        return 0;
    }
