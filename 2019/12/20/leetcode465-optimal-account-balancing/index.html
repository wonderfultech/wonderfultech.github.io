leetcode465 最优账单平衡 Optimal Account Balancing

解法一 回溯法

一开始所有人的钱都是0.

三元组 (x, y, z)  表示 x借给y共z元。

x借出z元，x的账户，从0变成-z元。

y借入z元，y的账户，从0变成z元。

第一步

首先对输入预处理，计算出，最终每个人，账户中的余额。 显然，余额等于0的就不用参与后序的运算了。最终目标，是将正余额，分配给负 余额 。让所有人的 余额 都变为0. 本题要求的就是最少的分配次数。

例如：[[0,6,7],[0,7,7],[1,4,5],[1,5,4],[2,5,2],[3,7,1]]

预处理后，变为下面的 余额 。格式为：人： 余额 。

3 : -1 2 : -2 6 : 7 0 : -14 7 : 8 1 : -9 4 : 5 5 : 6

将 余额 分成两组，正 余额 ：7 8 5 6

负 余额 ：-1 -2 -14 -9

目标就是将正 余额 分配到负 余额 。然后让正，负 余额 都变为0.

一种分配方法是，将8 6 拼起来分给-14，用了2次。

7分给-9，余-2，用了1次。

5分给-1 -2 -2，用了3次。至此，共用了6次，所有的 余额 都变为0.

用哈希表记录每个人的余额：

hash<人,余额>={};
for((x,y,z) in 输入){
    hash[x] -= z;
    hash[y] += z;
}

第二步

本题用贪心算法不能求出每种情况的最少分配次数。

也试过用网络流的思路分析问题，发现不容易找到其对应到网络流中的确切模型。

最终确定用暴力法求解。

对每个正 余额 ，将其分配到每个负 余额 。这样会导致正 余额 和负 余额 的列表发生变化。然后递归分配修改后的列表。当两个列表中的 余额 都是0时，就结束了。

接着上面的例子， 正 余额 ：7 8 5 6

负 余额 ：-1 -2 -14 -9

7可以分配到 -1 -2 -14 -9 中的任一个。

当7 分配到 -1时， 7+（-1）=6，6依然保留在 正 余额 列表中，正 余额 变为：6 8 5 6

负 余额 变为 ：0 -2 -14 -9

在修改后的正负余额列表上，递归计算。

当7 分配到 -14时， 7+（-14）=-7，-7保留在 负 余额 列表中，正 余额 变为：0 8 5 6

负 余额 变为 ：-1 -2 -7 -9

在修改后的正负余额列表上，递归计算。

pos;正余额列表
neg;负余额列表
s;正余额列表的偏移，要分配第几个正余额
z1;正余额列表中，0的个数
z2;负余额列表中，0的个数
c;正余额的分配次数
ans=INT_MAX;最少的分配次数
dfs(pos,neg,s,z1,z2,c){
    if(z1 == pos.length 且 z2 == neg.length){正负余额列表中都是0，说明结束了
        ans=min(ans,c);
        返回;
    }
    if(z1 == pos.length 或 z2 == neg.length) 正负余额列表中,有一个都是0，不用再分配，返回;
    //将pos[i]分配给neg[j]
    for(i in [s,pos.length-1]){
        for(j in [0,neg.length-1]){
            if(neg[j] == 0) pass掉;
            a=pos[i];
            b=neg[j];
            r=a+b;
            if(r>0){//pos[i]分配给neg[j]，pos[i]还有余额r，继续保留在pos[i].
                pos[i]=r;
                neg[j]=0;
                dfs(pos,neg,i,z1,z2+1,c+1);//neg中0增加1个
            }else if(r < 0){//pos[i]分配给neg[j]，neg[j]还差余额r，继续保留在neg[j].
                pos[i]=0;
                neg[j]=r;
                dfs(pos,neg,i+1,z1+1,z2,c+1);//pos中0增加1个
            }else{//pos[i]分配给neg[j]，刚好配平.
                pos[i]=0;
                neg[j]=0;
                dfs(pos,neg,i+1,z1+1,z2+1,c+1);//pos中0增加1个,neg中0增加1个
            }
            //恢复pos[i],neg[j]，保证下一轮分配前，pos和neg恢复原貌
            pos[i]=a;
            neg[j]=b;
        }
    }
}

代码：

    int ans;
    int minTransfers(vector<vector<int>>& transactions) {
        ans=INT_MAX;
        unordered_map<int,int> hash;
        for(int i=0;i<transactions.size();++i){
            vector<int>& v=transactions[i];
            hash[v[0]] -= v[2];
            hash[v[1]] += v[2];
        }
        unordered_map<int,int>::iterator it;
        int c1=0;
        int c2=0;
        vector<int> pos,neg;
        for(it=hash.begin();it!=hash.end();++it){
            if(it->second > 0) ++c1,pos.push_back(it->second);
            else if(it->second < 0) ++c2,neg.push_back(it->second);
        }
        
        dfs(pos,neg,0,0,0,0,0);

        return ans;
    }
    
    void dfs(vector<int>& pos,vector<int>& neg,
             int s1,int s2,
             int z1,int z2,int c){
        if(z1 == pos.size() && z2 == neg.size()){
            ans=min(ans,c);
            return ;
        }
        if(z1 == pos.size() || z2 == neg.size()) return;
        for(int i=s1;i<pos.size();++i){
            for(int j=s2;j<neg.size();++j){
                if(neg[j] == 0) continue;
                int a=pos[i];
                int b=neg[j];
                int r = a+b;
                if(r>0){
                    pos[i]=r;
                    neg[j]=0;
                    dfs(pos,neg,i,0,z1,z2+1,c+1);
                }else if(r<0){
                    pos[i]=0;
                    neg[j]=r;
                    dfs(pos,neg,i+1,0,z1+1,z2,c+1);
                }else{
                    pos[i]=0;
                    neg[j]=0;
                    dfs(pos,neg,i+1,0,z1+1,z2+1,c+1);
                }
                pos[i]=a;
                neg[j]=b;
            }
        }
    }
