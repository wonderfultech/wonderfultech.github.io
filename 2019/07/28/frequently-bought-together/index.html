frequently bought together 经常一起购买的商品

Posted on 2019年7月28日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

交易日志数据：

a b c
a b d
b c
b c

要得出这样的映射：商品->经常与其一起购买的商品。

用mapreduce解决这个问题。

map.py

#!/usr/bin/python
#coding=utf-8
import sys

def get_key(a,b):
    if a < b:
        return a+','+b
    else:
        return b+','+a

for line in sys.stdin:
    line = line.strip()
    segs = line.split()
    for i,x in enumerate(segs):
        for y in segs[i+1:]: 
            print('%s\t1'%(get_key(x,y)))

reduce.py

#!/usr/bin/python
#coding=utf-8
import sys

last_item = ''
last_count = 0

for line in sys.stdin:
    line = line.strip()
    segs = line.split()
    item = segs[0]
    count = long(segs[1])
    if last_item != item:
        if last_item != '':
            print('%s\t%d' % (last_item,last_count))
            last_item = ''
            last_count = 0
    last_item = item
    last_count += count

if last_item != '':
    print('%s\t%d' % (last_item,last_count))

提交脚本：

#!/bin/bash
#hadoop 2.7.6
JAR=${HADOOP_HOME}/share/hadoop/tools/lib/hadoop-streaming-2.7.6.jar
INPUT=/chap9_b/input/
OUTPUT=output
MAPPER=map.py
REDUCER=reduce.py
hadoop fs -rm -r  $OUTPUT
#进行连接
hadoop jar $JAR \
-D stream.map.input.ignoreKey=true \
-D stream.map.output.field.separator='\t' \
-D stream.num.map.output.key.fields=1 \
-D mapreduce.job.output.key.comparator.class=org.apache.hadoop.mapreduce.lib.partition.KeyFieldBasedComparator \
-D mapreduce.partition.keycomparator.options=-k1,1 \
-input $INPUT \
-output $OUTPUT \
-mapper $MAPPER \
-reducer $REDUCER \
-file $MAPPER \
-file $REDUCER \
-partitioner org.apache.hadoop.mapred.lib.KeyFieldBasedPartitioner

执行结果：

a,b 2
a,c 1
a,d 1
b,c 3
b,d 1

由这样的结果，可以得出每个商品与其常买的商品列表。
