customer who buy the item also buy 买过该商品的人还买了哪些

Posted on 2019年7月28日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

在电商网站上，当你购买一件商品时，网站会推荐“购买过该商品的顾客还购买了哪些商品”。吸引用户接着下单购买。

从用户的购买日志开始。日志格式为：user item1 item2 item3......

从这份日志中，分析出买了某种商品的人还买了哪些商品。

算法分成两个阶段：

阶段1：收集每个用户购买的所有商品。

阶段2：挑出买了商品A后，还买了的其它商品的top5个。

阶段一

map.py

恒等映射。

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    print(line)

reduce.py

收集每个用户的商品列表。

#!/usr/bin/python
#coding=utf-8
import sys

last_user = ''
last_items = []

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    user = segs[0]
    items = segs[1:]
    if user != last_user:
        if last_user != '':
            print('%s\t%s'%(last_user,' '.join(last_items))) 
            last_user = '' 
            last_items = []
    last_user = user
    for x in items:
        last_items.append(x)

if last_user != '':
    print('%s\t%s'%(last_user,' '.join(last_items)))

阶段二

map2.py

统计每个用户，各个商品的购买数量。并且发射每个商品和这个用户的商品购买统计结果。

#!/usr/bin/python
#coding=utf-8
import sys

for line in sys.stdin:
    line = line.strip()
    mm={}
    segs = line.split()
    user = segs[0]
    items = segs[1:]
    #统计item的个数
    for x in items:
        if x not in mm:
            mm[x] = 0
        mm[x] = mm[x] + 1
    #对mm进行串行化
    s=[]
    for item,count in mm.items():
        s.append('%s,%d'%(item,count))
    #对每个item，发出mm
    for item in mm.keys():
        print('%s\t%s'%(item,';'.join(s)))

reduce2.py

收集每个用户发射出的（商品，购买统计结果）。将相同商品的购买统计结果累加起来。将商品与用户切分开来。最后，发射每个商品，及其购买该商品的用户还会购买的商品的统计个数最高的前5个。

#!/usr/bin/python
#coding=utf-8
import sys

last_item = ''
last_mm = []

def get_mm(s):
    '''
    从字符串中还原出mm。
    '''
    r={}
    segs = s.split(';')
    for x in segs:
        kv = x.split(',')
        item = kv[0]
        count = long(kv[1])
        r[item] = count
    return r

def add_mm():
    '''
    合并各个mm中item的计数到一个mm中。
    并输出item关联的top5的item
    '''
    res={}
    for x in last_mm:
        r = get_mm(x) 
        for y,c in r.items():
            if y not in res:
                res[y] = 0
            res[y] = res[y] + c
    #选出计数最高的top5
    top_list = [(k,v) for k,v in res.items()]
    top_list.sort(key=lambda e:e[1])
    out=[]
    for x in top_list[:5]:
        out.append(x[0]+','+str(x[1]))
    print('%s\t%s'%(last_item,';'.join(out)))

for line in sys.stdin:
    line = line.strip()
    #print(line)
    segs = line.split()
    item = segs[0]
    mm = segs[1]
    if item != last_item:
        if last_item != '':
            add_mm()
            last_item = item
            last_mm = []
    last_item = item
    last_mm.append(mm)

if last_item != '':
    add_mm()
