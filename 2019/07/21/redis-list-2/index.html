leetcode 数据库题目全部题解

ziplist编码：

linkedlist编码:

列表对象支持的各种函数：其中有Command的是支持命令的实现函数。

listTypeTryConversion（subject,value）：对value字符串进行检查，如果超过了ziplist的最大长度，则要将subject从ziplist转换为linked_list。

listTypePush(subject,value)：将value添加到表头或表尾。
	调用listTypeTryConversion;
	如果subject的编码是ziplist并且超过了ziplist的最大长度，则要将subject从ziplist转换为linked_list。
	如果subject的编码是ziplist，
		解码value到字符串或整数，ziplist只能装字符串或整数。
		添加到ziplist的头或尾；
	如果subject的编码是linked_list，
		添加到linked_list的头或尾；

listTypePop(subject):从链表的头部或尾部弹出一个元素。并返回元素对象。
	如果subject的编码是ziplist，
		ziplistIndex定位元素；
		ziplistGet取出元素v；
		为元素v创建对象value；
		从ziplist中删除元素v；
	如果subject的编码是linked_list，
		listNodeValue取出头尾节点的值value；
		listDelNode删除头尾节点；
	返回节点；

listTypeLength(subject):返回列表的元素个数。
	如果subject的编码是ziplist，返回ziplistlen；
	如果subject的编码是linked_list，返回listLength；

listTypeInitIterator（subject，index，direction）：在列表index位置处创建方向direction的迭代器。
	如果subject的编码是ziplist，ziplistindex定位元素p，返回此处的迭代器；
	如果subject的编码是linked_list，listIndex定位元素p，返回此处的迭代器；

listTypeReleaseIterator：释放迭代器。

listTypeNext（iter，entry）：entry指向当前节点，将迭代其iter指向下一个节点。
	如果subject的编码是ziplist，entry记录当前节点，ziplistNext或ziplistPrev滑动迭代器；
	如果subject的编码是linked_list，entry记录当前节点，迭代器指针向next或prev滑动迭代器；

listTypeGet（entry）：取出entry的节点的值。
	如果列表的编码是ziplist，
		ziplistGet取出值，并为其创建对象；
	如果列表的编码是linked_list，
		listNodeValue取出值，并增加其引用计数；

listTypeInsert（entry，value，where）：将value插入到列表节点entry的前或后。
	如果列表的编码是ziplist，
		返回value未编码的值，ziplistInsert插入；
	如果列表的编码是linked_list，
		listInsertNode插入节点，并增加其引用计数；

listTypeEqual（entry，o）：比较节点的值与对象o。
	如果列表的编码是ziplist，
		ziplistCompare比较；
	如果列表的编码是linked_list，
		equalStringObjects比较对象值；

listTypeDelete（entry）：删除节点。
	如果列表的编码是ziplist，
		ziplistDelete删除节点，并更新迭代器；
	如果列表的编码是linked_list，
		listDelNode删除节点，并更新迭代器；

listTypeConvert：将列表由ziplist换成linked_list。
	创建ziplist的迭代器；
	遍历迭代器，listTypeGet取出值，并listAddNodeTail插入到链表中。
	释放原来的ziplist；
	
pushGenericCommand(client,where):添加输入值到列表的底层函数。
	取处key的列表对象obj；
	遍历所有输入值：
		将每个值编码成值对象val;
		如果obj不存在，创建新列表对象，并与key关联；
		将值val，listTypePush，加入列表的头部或尾部；

lpushCommand(client):LPUSH key value [value…]
	pushGenericCommand添加到列表头部；
rpushCommand(client):RPUSH key value [value…]
	pushGenericCommand添加到列表尾部；
pushxGenericCommand(client,refval/*参照值*/,val/*新值*/,where):列表push底层函数。
	取出key对应的列表对象subject，不存在返回；
	如果refval：（此时LINSERT命令）
		listTypeTryConversion是否需要将ziplist转换未linked_list；
		定位refval在列表中的位置，将val插入到refval前面或后面；
		检查是否需要将ziplist转换未linked_list
	其它：（LPUSHX或RPUSHX命令）
		listTypePush将val添加到列表首部或尾部；
lpushxCommand（client）：LPUSHX key value。当key存在时，才在头部插入值。
	编码value对象；
	pushxGenericCommand
rpushxCommand（client）：RPUSHX key value。当key存在时，才在尾部插入值。
	编码value对象；
	pushxGenericCommand
insertCommand（client）：LINSERT key before|after pivot value。在参考值pivot的before或after位置插入新值value。
	编码参考值pivot对象；
	pushxGenericCommand
llenCommand：LLEN key。列表的长度。
	listTypeLength返回列表长度。
lindexCommand：LINDEX key index。列表位置index处的元素。
	如果列表的编码是ziplist，
		ziplistIndex定位到index处的元素；
		ziplistGet取出元素值；
		创建字符串或整数对象，返回；
	如果列表的编码是linked_list，
		listIndex定位到index处的节点；
		取出节点处的值，并返回；
lsetCommand：LSET key index value。设置index位置处的值为value。
	取出key处的列表对象subject；
	如果列表的编码是ziplist，
		index越界，返回错误；
		ziplistIndex定位到index处的元素；
		ziplistDelete删除旧元素；
		ziplistInsert插入新元素value；
	如果列表的编码是linked_list，
		index越界，返回错误；
		listIndex定位到index处的节点；
		删除旧值；
		listNodeValue赋新值；
popGenericCommand：pop底层函数。
	取出key处的列表对象subject；
	listTypePop弹出列表元素；
lpopCommand:LPOP key。弹出第一个元素。
rpopCommand:RPOP key。弹出最后一个元素。
	popGenericCommand；
lrangeCommand：LRANGE key start stop。取出列表位置[start，stop]中的元素。
	取出key的列表对象；
	如果列表的编码是ziplist，
		ziplistIndex定位到start处的元素；
		ziplistGet连续取出确定个元素值，逐个答复；
	如果列表的编码是linked_list，
		listIndex定位到index处的节点；
		连续取出确定个节点的值，逐个答复，
ltrimCommand：LTRIM key start stop。剔除列表中的元素，只保留[start,stop]范围内的元素。
	取出key的列表对象；
	如果列表的编码是ziplist，
		ziplistDeleteRange删除左端和右端的元素；
	如果列表的编码是linked_list，
		listDelNode删除左端和右端的元素；
lremCommand：LREM key count value。
		Count>0，从头部开始删除count个value值；
		Count<0，从尾部开始删除count个value值；
		Count=0，删除全部value值；
	取出key的列表对象；
	构造迭代器；
	遍历迭代器，查找值value，并listTypeDelete删除与value相等的元素；
rpoplpushHandlePush（client，dstkey，dstobj，value)：将value放到dstobj的头部。
	如果dstobj不存在，数据库中存储dstkye-dstobj；
	将value，压入dstobj的首部；
rpoplpushCommand:RPOPLPUSH srckey dstkey。将srckey列表的最后一个元素，取出，并压入dstkey列表的首部。
	取出srckey的列表对象，如果没有就返回；
	listTypePop从列表中弹出对象；
	rpoplpushHandlePush将对象压入目标列表；

blockingPopGeneriCommand（client，where）：阻塞RPOP/LPOP。
	遍历每个key：
		取出每个key的列表对象；
		如果有一个列表对象不为空，弹出一个元素并返回；
	如果所有的key都为空或不存在，就阻塞timeout的时间；
blpopCommand：BLPOP key [key…] timeout。阻塞弹出第一个元素。
brpopCommand：BRPOP key [key…] timeout。阻塞弹出第一个元素。
	blockingPopGeneriCommand完成；
brpoplpushCommand:BRPOPLPUSH srckey dstkey timeout。将srckey列表的最后一个元素，取出，并压入dstkey列表的首部。如果srckey不存在或其列表为空时，阻塞timeout的时间。
	取出srckey的列表对象，如果srckey不存在或其列表为空时，阻塞timeout的时间；
	否则执行rpoplpushCommand；

支持的命令：

get函数：
	lindexCommand：LINDEX key index。列表位置index处的元素。
	lrangeCommand：LRANGE key start stop。取出列表位置[start，stop]中的元素。
set函数：
	lsetCommand：LSET key index value。设置index位置处的值为value。
	insertCommand（client）：LINSERT key before|after pivot value。在参考值pivot的before或after位置插入新值value。
	lpushCommand(client):LPUSH key value [value…]
	rpushCommand(client):RPUSH key value [value…]
	lpushxCommand（client）：LPUSHX key value。当key存在时，才在头部插入值。
	rpushxCommand（client）：RPUSHX key value。当key存在时，才在尾部插入值。
	rpoplpushCommand:RPOPLPUSH srckey dstkey。将srckey列表的最后一个元素，取出，并压入dstkey列表的首部。
	阻塞：
	brpoplpushCommand:BRPOPLPUSH srckey dstkey timeout。将srckey列表的最后一个元素，取出，并压入dstkey列表的首部。如果srckey不存在或其列表为空时，阻塞timeout的时间。
删除函数：
	lpopCommand:LPOP key。弹出第一个元素。
	rpopCommand:RPOP key。弹出最后一个元素。
	ltrimCommand：LTRIM key start stop。剔除列表中的元素，只保留[start,stop]范围内的元素。
	lremCommand：LREM key count value。
	阻塞：
	blpopCommand：BLPOP key [key…] timeout。阻塞弹出第一个元素。
	brpopCommand：BRPOP key [key…] timeout。阻塞弹出第一个元素。
杂项函数：
llenCommand：LLEN key。列表的长度。
