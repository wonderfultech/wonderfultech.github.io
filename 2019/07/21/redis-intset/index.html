redis intset 整数集合

Posted on 2019年7月21日2019年7月23日by jasonpeng

leetcode 数据库题目全部题解

intset其实就是字节数组，用16位，32位，64位解析数据，解析的值有序，不重复（所以叫整数集合）。可能会有编码方式的升级，但是不支持降级。

//定义
typedef struct intset {
uint32_t encoding;//编码
uint32_t length;//元素个数
int8_t contents[];//存储空间
} intset;
//编码类型
#define INTSET_ENC_INT16 (sizeof(int16_t))
#define INTSET_ENC_INT32 (sizeof(int32_t))
#define INTSET_ENC_INT64 (sizeof(int64_t))

intset支持的算法。

get：取指定位置pos处的值；
set：设置位置pos处的值；
search：二分查找；

upgradeAndAadd:编码升级，并在升级后的数组前或后添加新元素，因为新值要么比全部值大或小；
	用新编码resize数组（原始数据得以保留）；
	用旧编码从数组中取处原来的值v，再以新编码填充到预定位置，从后向前；
	在数组前或后添加新值；

moveTail：移动尾部的元素到指定位置
Add：添加新元素v
	如果需要升级编码，调用upgradeAndAadd；
	否则：
		在数据中二分查找v；
		找到v，添加失败返回；
		未找到v，记录插入位置pos：
			resize数组，增加一个元素空间；
			用moveTail后移一个元素的位置；
			在pos处放置新元素v;

Remove：删除元素v
	如果v的编码方式小于等于数组的编码方式，二分查找v；
		找到v，moveTail前移一个元素的位置，删除元素v；
	其它，返回
Find：查找元素v
	如果v的编码方式小于等于数组的编码方式，二分查找v；
Random：返回随机元素
在长度中随机选一个位置，取出元素；
