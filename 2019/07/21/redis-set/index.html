Redis set 集合

Posted on 2019年7月21日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

intset编码：

散列表编码：键是字符串对象，也是集合的元素，值全为NULL。

支持的命令：

set函数：
	saddCommand：SADD key member [member ...]。向集合中添加多个元素。
get函数：
	srandmemberWithCountCommand：SRANDMEMBER key count。随机返回count个元素。
	srandmemberCommand：SRANDMEMBER key [count]。随机返回count个元素。
	sscanCommand:SSCAN key cursor [MATCH pattern] [COUNT count
	
删除函数：
	sremCommand：SREM key member [member ...]。从集合中删除多个元素。
	spopCommand：SPOP key [count]。随机删除一个元素并返回。
	
移动函数：
	smoveCommand：SMOVE source destination member。将元素member从source移动到destination。

成员检测函数：
	sismemberCommand：SISMEMBER key member。元素member是否在集合中。

交集函数：
	sinterCommand：SINTER key [key ...]。计算多个集合的交集。
	sinterStoreCommand：SINTERSTORE destination key [key ...]。计算多个集合的交集，并存储在dest集合中。
	
并集函数：
	SunionCommand：SUNION key [key ...]。集合的并集。
	SunionStoreCommand：SUNIONSTORE destination key [key ...]。集合的并集，将结果存储到dest集合。

差集函数：
	SdiffCommand：SDIFF key [key ...]。集合的差集。
	SdiffStoreCommand：SDIFFSTORE destination key [key ...]。集合的差集，将结果存储到dest集合。

杂项函数：
scardCommand：SCARD key。集合基数。

支持的函数：

setTypeCreate（value）：返回保存value值的集合。
	如果value能表示成整数，则createIntsetObject创建intset；
	否则createSetObject创建以dict字典为底层的集合；
setTypeAdd（subject，value)：向集合中添加value。
	如果subject是dict类型，dictAdd；
	如果subject是intset类型:
		如果value能表示成整数，intsetAdd；
		否则，先将intSet转换为dict字典，再dictAdd；
setTypeRemove（setobj，value）：从集合中删除value。
	如果subject是dict类型，dictDelete删除value；
	如果subject是intset类型，
		如果value能表示成整数，intsetRemove；
setTypeIsMember（subject，value）：value是否在subject中。
	如果subject是dict类型，dictFind删除value；
	如果subject是intset类型，
		如果value能表示成整数，intsetFind；
setTypeInitIterator（subject）：创建集合迭代器。
	如果subject是dict类型，dictGetIterator；
	如果subject是intset类型，下标0就是迭代器；
setTypeReleaseIterator：释放迭代器。
	如果subject是dict类型，dictReleaseIterator；
setTypeNext：返回迭代器当前节点的值，并将迭代器指向下一个元素。
	如果subject是dict类型，dictGetKey取键，dictNext更新迭代器；
	如果subject是intset类型，intsetGet取出元素；
setTypeNextObject：返回迭代器当前节点的新值，并将迭代器指向下一个元素。
	setTypeNext取出新值；
	如果subject是dict类型，增加新值的引用计数；
	如果subject是intset类型，复制一个整数值并返回；
setTypeRandomElement：
	如果subject是dict类型，dictGetRandomKey，dictGetKey返回随机键，
	如果subject是intset类型，intsetRandom返回随机整数；
setTypeSize：
	如果subject是dict类型，dictSize返回；
	如果subject是intset类型，intsetLen返回；
setTypeConvert:将intset编码的集合转换成哈希表。
	dictCreate创建新字典；
	dictExpand将新字典扩展到与intset同样的大小；
	setTypeInitIterator初始化迭代器；
	迭代器遍历，将每个整数转成字符串，用dictAdd添加到字典中；
	释放intset集合；
saddCommand：SADD key member [member ...]。向集合中添加多个元素。
	lookupKeyWrite取出集合对象set；
	如果set不存在，创建一个新集合对象，并添加到字典中；
	对每个输入的元素：
		tryObjectEncoding编码元素；
		setTypeAdd插入集合；
sremCommand：SREM key member [member ...]。从集合中删除多个元素。
	lookupKeyWriteOrReply取出集合对象；
	对每个输入的元素：
		setTypeRemove删除元素；
smoveCommand：SMOVE source destination member。将元素member从source移动到destination。
	lookupKeyWrite取出源集合，不存在返回；
	lookupKeyWrite取出目标集合，不存在，创建新集合；
	setTypeRemove从源集合移除元素；
	setTypeAdd将元素插入目标集合；
sismemberCommand：SISMEMBER key member。元素member是否在集合中。
	lookupKeyReadOrReply取出集合对象；
	tryObjectEncoding编码元素；
	setTypeIsMember检查元素是否在集合中，返回；
scardCommand：SCARD key。集合基数。
	lookupKeyReadOrReply取出集合对象；
	返回setTypeSize；
spopCommand：SPOP key [count]。随机删除一个元素并返回。
	lookupKeyWriteOrReply取出集合对象；
	setTypeRandomElement随机取出一个元素；
	如果subject是dict类型，setTypeRemove删除元素；
	如果subject是intset类型，intsetRemove删除元素；
srandmemberWithCountCommand：SRANDMEMBER key count。随机返回count个元素。
	如果count为负数，返回的集合中可以有重复元素，setTypeRandomElement随机取-count个元素，返回；
	如果count>=集合基数，suniouDiffGenericCommand返回整个集合；
	其它，创建一个新字典dict对象d；
	当count少于集合基数，但是接近于集合基数时，从集合中随机取出count个元素非常低效；
		setTypeInitIterator创建迭代器；
		遍历迭代器，将集合中的元素全部添加到集合d中；
		如果集合基数>count:
			dictGetRandomKey随机取出一个元素；
			dictDelete删除此元素；
	当count比集合基数小很多时，
		setTypeRandomElement随机取出元素；
		dictAdd添加到d中；
srandmemberCommand：SRANDMEMBER key [count]。随机返回count个元素。
	如果带有count参数，封装srandmemberWithCountCommand；
	lookupKeyReadOrReply取出集合对象；
	setTypeRandomElement随机取出元素；
qsortCompareSetsByCardinality(s1,s2)：比较函数。比较两个集合(s1,s2)的基数大小。
qsortCompareSetsByRevCardinality(s1,s2)：比较函数。比较两个集合(s2,s1)的基数大小。
sinterGenericCommand（client，setkeys，setnum，dstkey）：
	取出setnum个集合对象，放入集合对象数组sets；
	对集合对象数组sets，按集合基数，从小到大排序；
	为排在第一位的集合（基数最小的集合）创建迭代器；
	对每个迭代器的元素：
		只要剩余的集合中有一个不含有此元素，就丢掉此元素，否则收集此元素；
sinterCommand：SINTER key [key ...]。计算多个集合的交集。
sinterStoreCommand：SINTERSTORE destination key [key ...]。计算多个集合的交集，并存储在dest集合中。
	sinterGenericCommand封装；
sunionDiffGenericCommand（client，setkeys，setnum，dstkey）：
	取出setnum个集合对象，放入集合对象数组sets；
	决定采用那个算法实现集合差：
		algo_one_work为setnum x sets[0] 的基数；
		algo_two_work所有集合的基数之和；
		如果algo_one_work/2  <= algo_two_work，则选择算法1，否则选择算法2;（等价于，如果集合0的基数的一半小于等于平均基数，采用算法1）；
		如果采用算法1，将集合对象数组，按集合基数的逆序排列；
	创建结果集；
	集合并：
		遍历所有集合，将元素添加进结果集；
	集合差算法1：
		对集合0中的每个元素e；
		如果e不在其它所有集合中，则e添加进结果集；
	集合差算法2：
		将集合0中的元素全部加入结果集；
		遍历其它全部集合的每个元素e，如果e在结果集中，则将e从结果集中删除；
SunionCommand：SUNION key [key ...]。集合的并集。
SunionStoreCommand：SUNIONSTORE destination key [key ...]。集合的并集，将结果存储到dest集合。
	sunionDiffGenericCommand封装；
SdiffCommand：SDIFF key [key ...]。集合的差集。
SdiffStoreCommand：SDIFFSTORE destination key [key ...]。集合的差集，将结果存储到dest集合。
	sunionDiffGenericCommand封装；
sscanCommand:SSCAN key cursor [MATCH pattern] [COUNT count]。（与SCAN命令一块学习）
