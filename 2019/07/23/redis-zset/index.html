Redis zset 有序集合

Posted on 2019年7月23日2019年7月23日by jasonpeng

leetcode 数据库题目全部题解

ziplist编码：成员和分值排在一起。按分值从小到大排序。

skiplist编码：包含一个字典和一个skiplist。

字典：为有序集合创建成员->分值映射。

skiplist：score保存分值，object保存成员。按分值有序排列节点。

成员都是字符串对象，分值都是double类型。

支持的函数：

zzlCompareElements（eptr，cstr）。将有序集中的字符串eptr与字符串cstr比较。
	ziplistGet取出eptr的字符串值；
	memcmp，返回；
zzlength：返回有序集合中的元素个数。
	ziplistLen/2，返回；
zzlNext（eptr，sptr）：根据eptr和sptr，移动到下一个entry。
	ziplistNext；
	ziplistNext；
zzlPrev（eptr，sptr）：根据eptr和sptr，移动到前一个entry。
	ziplistPrev；
	ziplistPrev；
zzlIsInRange（zl，range）：zl中是否有range范围内的节点。
	ziplistIndex；
	zzlGetScore；
	zslValueGteMin；
	ziplistIndex；
	zzlGetScore；
	zslValueLteMax；
zzlFirstInRange（zl，range）：返回zl中第一个在range范围内的节点。
	zzlIsInrange判断是否在range内，不在返回；
	对ziplist中的每个节点：
		zzlGetScore取其分值score；
		如果score满足zslValueGteMin且zslValueLteMax，则返回其节点；
zzlLastInRange（zl，range）：返回zl中最后一个在range范围内的节点。
	方法与zzlFirstInRange类似；
zzlLexValueGteMin（p，spec）：
zzlLexValueLteMax（p，spec）：
zzlIsInLexRange（zl，range）：
zzlFirstInLexRange（zl，range）：
zzlLastInLexRange（zl，range）：
zzlFind（zl，ele，score）：从ziplist中查找ele，返回分数score。
	对ziplist中的每个元素：
		ziplistNext指向分数；
		ziplistCompare对比节点与ele，如果找到ele，zzlGetScore返回分值；
zzlDelete（zl，eptr）：从ziplist中删除eptr指向的field-score。
	ziplistDelete删除field节点；
	ziplistDelete删除score节点；
zzlInsertAt（zl，eptr，ele，score）：将ele-score添加到eptr所指向的节点之前。
	集合中成员排列：| member-1 | score-1 | member-2 | score-2 | ... | member-N | score-N |
	如果eptr为空，
		ziplistPush添加ele；
		ziplistPush添加score；
	否则，在eptr前
		ziplistInsert插入ele；
		ziplistInsert插入socre；
zzlInsert（zl，ele，score）：将ele-score插入ziplist。
	对ziplist中的每个member-score：
		zzlGetScore取出分值s；
		如果s>score，在此是第一个大于score的member；
			zzlInsertAt插入ele-score，返回；
		如果s==score，再比较member与ele的值，判断插入位置；
zzlDeleteRangeByScore（zl，range）：删除ziplist中在范围range中的元素。
	zzlFirstInRange找第一个符合范围的节点，找不到返回；
	一直删除节点，找到不在range范围内；
zzlDeleteRangeByLex（zl，range）：
zzlDeleteRangeByRank（zl，start，end）：删除[start,end]索引范围内的元素。
	ziplistDeleteRange封装；
zsetLength：zset的长度。
	如果是ziplist编码，返回zzllength；
	如果是skiplist编码，返回zsl的长度；
zsetConvert：zset的ziplist编码与skiplist编码相互转换。
	从ziplist转换成skiplist：
		创建有序集合zset；
		创建字典dict，跳跃表zsl；
		ziplist集合中成员排列：| member-1 | score-1 | member-2 | score-2 | ... | member-N | score-N |
		对ziplist中的每个成员：
			ziplistGetScore取出score；
			ziplistGet取出member；
			zslInert将score-member插入skiplist；
			dictAdd将member-score添加到字典dict中；
		释放原先的ziplist；
	从skiplist转换成ziplist：
		创建新的ziplist；
		释放字典dict，不需要；
		对skiplist中的每个节点：
			zzlInsertAt将member-score添加到ziplist；
			释放节点；
		释放skiplist；
zaddGenericCommand（client，incr）：
	lookupKeyWrite取出zset对象；
	如果对象不存在，为key创建一个新的；
	对输入的每对member-score：
		如果是ziplist编码：
			zzlFind查找成员；
			如果成员存在：
				增加分值或更新分值；
			不存在：
				zzlInsert添加成员-分值；
				并检查是否需要将编码转成skiplist；
		如果是skiplist编码：
			dictFind查找成员；
			如果成员存在：
				取出成员dictGetKey；
				取出分值dictGetVal；
				增加分值或更新分值；
			不存在：
				zslInsert将分值-成员添加进skiplist；
				dictAdd将成员-分值添加进字典dict；
zaddCommand：ZADD key [NX|XX] [CH] [INCR] score member [score member ...]。添加分值-成员到有序集合。
	zaddGenericCommand封装；
zincrbyCommand:ZINCRBY key increment member。成员的分值增加incr。
	zaddGenericCommand封装；
zremCommand：ZREM key member [member ...]。删除成员。
	lookupKeyWriteOrReply取出对象；
	如果是ziplist编码：
		对每个输入的成员：
			zzlFind查找成员，找到zzlDelete删除成员。
	如果是skiplist编码：
		对每个输入的成员：
			dictFind查找成员；
			找到：
				将分值用zslDelete从skiplist中删除；
				dictDelete从字典中删除成员；
zemrangeGenericCommand：
	确定范围；
	lookupKeyWriteOrReply取出对象；
	如果是ziplist编码：
		删除范围内的值：
			zzlDeleteRangeByRank
			zzlDeleteRangeByScore
			zzlDeleteRangeByLex
	如果是skiplist编码：
		删除范围内的值：
			zslDeleteRangeByRank
			zslDeleteRangeByScore
			zslDeleteRangeByLex
zremrangebyrankCommand：ZREMRANGEBYRANK key start stop。删除指定排名范围内的成员。
zremrangebyscoreCommand：ZREMRANGEBYSCORE key min max。删除指定分值范围内的成员。
zremrangebylexCommand：ZREMRANGEBYLEX key min max。删除指定字典序范围内的成员。
	zremrangeGenericCommand封装；
zuiInitIterator：初始化集合或有序集合的迭代器。
zuiClearIterator：清除迭代器。
zuiLength：被迭代对象的元素个数。
zuiNext：取出当前迭代器的值，并迭代器指向下一个元素。
zuiLongLongFromValue：从val中取出long long值。
zuiObjectFromValue：从val中创建对象。
zuiBufferFromValue：从val中取出字符串。
zuiFind：从迭代器所在的对象中查找指定值。
zuiCompareByCardinality：比较两个迭代器所在对象的基数。
zunionInterAggregate(*target, val, aggregate)：聚集函数MAX，MIN，SUM。
zunionInterGenericCommand：
	为每个输入的key：
		lookupKeyWrite取出对象；
		为对象创建迭代器；
	取出选项参数：权重，聚集类型；
	对所有有序集合，按集合基数由小到大排序；
	创建结果集对象；
	如果是ZINTERSTORE命令：
		zuiInitIterator为基数最小的创建迭代器；
		对迭代器中的每个节点ele：
			如果ele在所有其它集合中，那么zunionInterAggregate计算加权聚集；
			结果存储到结果集中；
	如果是ZUNIONSTORE命令：
		对每个输入的集合：
			对集合中的每个元素ele：
				如果ele已经在结果集中，跳过；
				将ele在剩余的集合中，zunionInterAggregate计算加权聚集；
				结果存储到结果集中；
zunionstoreCommand：ZUNIONSTORE destination numkeys key [key ...] [WEIGHTS weight] [SUM|MIN|MAX]。并集。
zinterstoreCommand：ZINTERSTORE destination numkeys key [key ...] [WEIGHTS weight] [SUM|MIN|MAX]。交集。
	zunionInterGenericCommand封装；
zrangeGenericCommand：
	取出范围参数；
	lookupKeyReadOrReply取出集合对象；
	如果是ziplist编码：
		ziplistIndex定位开始元素；
		ziplistGet连续取出元素；
	如果是skiplist编码：
		zslGetElementByRank定位开始元素；
		skiplist指针遍历；
zrangeCommand：ZRANGE key start stop [WITHSCORES]。返回范围内的值。
zrevrangeCommand：ZREVRANGE key start stop [WITHSCORES]。返回范围内（从大到小）的值。
	zrangeGenericCommand封装；
genericZrangebyscoreCommand：
	取出范围参数；
	lookupKeyReadOrReply取出集合对象；
	如果是ziplist编码：
		zzlFirstInRange或zzlLastInRange定位开始节点；
		返回范围内的元素zslValueGteMin或zslValueLteMin；
	如果是skiplist编码：
		zslFirstInRange或zslLastInRange定位开始节点；
		返回范围内的元素zslValueGteMin或zslValueLteMin；
zrangebyscoreCommand:ZRANGEBYSCORE key min max [WITHSCORES] [LIMIT offset count]。返回分数范围内的值。
zrevrangebyscoreCommand：ZREVRANGEBYSCORE key max min [WITHSCORES] [LIMIT offset count]。返回分数范围内（逆向）的值。
	genericZrangebyscoreCommand封装；
zcountCommand：ZCOUNT key min max。分值范围内元素的个数。
	取出范围参数；
	lookupKeyReadOrReply取出集合对象；
	如果是ziplist编码：
		zzlFirstInRange定位开始节点；
		zzlGetScore取分值；
		记录分值在范围内的个数；
	如果是skiplist编码：
		zslFirstInRange定位开始节点；
		zslGetRank取分值的排名计算个数；
zlexcountCommand：ZLEXCOUNT key min max。字典序范围内的个数。
	取出范围参数；
	lookupKeyReadOrReply取出集合对象；
	如果是ziplist编码：
		zzlFirstInLexRange定位开始节点；
		记录成员在范围内的个数；
	如果是skiplist编码：
		zslFirstInLexRange定位开始节点；
		记录成员在范围内的个数；
genericZrangebylexCommand：字典序范围的遍历。与genericZrangebyscoreCommand：类似不再赘述。
zrangebylexCommand：ZRANGEBYLEX key min max [LIMIT offset count]。字典序范围内的元素。
zrevrangebylexCommand：ZREVRANGEBYLEX key min max [LIMIT offset count]。字典序范围内（逆序）的元素。
	genericZrangebylexCommand封装；
zcardCommand：ZCARD key。返回集合基数。
	zsetLength；
zscoreCommand：ZSCORE key member。返回成员的分值。
	lookupKeyReadOrReply取出集合对象；
	如果是ziplist编码：
		zzlFind找到成员，返回分值；
	如果是skiplist编码：
		dictFind找到成员，返回分值；
zrankGenericCommand：
	lookupKeyReadOrReply取出集合对象；
	如果是ziplist编码：
		遍历元素ele，用ziplistCompare和zzlNext计算排名；
	如果是skiplist编码：
		dictFind找到成员；
		取出其分值；
		从skiplist中用zslGetRank取出排名；
zrankCommand：ZRANK key member。返回成员的分值排名；
zrevrankCommand：ZREVRANK key member。返回成员的分值排名（逆向）；
    zrankGenericCommand；
zscanCommand:以后再学。
