Redis database 数据库

Posted on 2019年7月23日2019年8月30日by jasonpeng

leetcode 数据库题目全部题解

server有多个数据库；client可以选择要使用的数据库。

数据库结构：

typedef struct redisDb {
	dict *dict; /* The keyspace for this DB */ 键字典
	dict *expires; /* Timeout of keys with a timeout set */ 有超时值的键字典
	dict *blocking_keys; /* Keys with clients waiting for data (BLPOP) */ 被阻塞键的字典
	dict *ready_keys; /* Blocked keys that received a PUSH */ 就绪键字典
	dict *watched_keys; /* WATCHED keys for MULTI/EXEC CAS */ 被监视的键字典
	struct evictionPoolEntry *eviction_pool; /* Eviction pool of keys */
	int id; /* Database ID */
	long long avg_ttl; /* Average TTL, just for stats */
} redisDb;

键空间——dict字段，是字典：键值对。键——数据库键，字符串对象。值——字符串对象，列表对象，哈希表对象，集合对象，有序集合对象。常见的操作是SET，DEL，GET。

过期字典——expires字段，是字典：键值对。键——数据库键。值——过期时间。常见的操作有PEXPIREAT，PERSIST，TTL。

过期删除策略：惰性删除与定期删除，相结合。

  1. 惰性删除——操作键时，先判断是否过期，过期先删除，再做其它动作。Db.c/expireIfNeeded函数完成。
  2. 定期删除——限制删除时长和频率。Redis.c/activeExpireCycle函数实现。
    1. 多次遍历数据库；
    2. 从expires字典中，随机选择键，如果过期就删除。
  3. AOF，RDB和复制机制下过期键的处理：再补充！！！

内部函数：

lookupKey（db，key）：从db中取出key的值。
	在db->dict中用dictFind查找key；
	找到返回值，否则返回NULL；
lookupKeyRead（db，key）：为执行读取操作而从db中取出key的值。
	expireIfNeeded过期删除key；
	lookupKey取值；
	值为NULL更新更新不命中信息，否则命中信息;
lookupKeyWrite（db，key）：为执行写入操作而从db中取出key的值。
	expireIfNeeded过期删除key；
	lookupKey取值，返回；
lookupKeyReadOrReply（client，db，key）：为执行读取操作而从db中取出key的值。key不存在答复客户端。
	lookupKeyRead取值；
	值为NULL答复客户端，否则返回值；
lookupKeyWriteOrReply（client，db，key）：为执行写入操作而从db中取出key的值。key不存在答复客户端。
	lookupKeyWrite取值；
	值为NULL答复客户端，否则返回值；
dbAdd（db，key，val）：将key-val添加到db中，如果key已存在，则停止。
	dictAdd（dict，key，val）；
dbOverwrite（db，key，val）：用val覆盖key的旧值，如果key不存在，则停止。
	dictFind（dict，key）查找key；
	不存在停止；
	dictReplace覆盖dict中的旧值；
setKey（db，key，val）：向数据库中添加key-val。
	lookupKeyWrite取出key的值；
	如果值为NULL：
		dbAdd添加key-val；
	不为NULL：
		dbOverwrite用val覆盖旧值；
	增加val的引用计数；
	removeExpire删除过期的键；
	signalModifiedKey通知监视key的客户端；
dbExists（db，key）：检查key是否在数据库中。
	dictFind查找key；
dbRandomKey（db）：随机从数据库中选择一个键。
	while：
		dictGetRandomKey随机取一个节点de；
		如果de为NULL，返回NULL；
		dictGetKey取出键key；
		dictFind（expires）检查key是否有过期时间：
			expireIfNeeded删除过期键，重复；
		返回key；
dbDelete（db，key）：删除键。
	dictDelete（expires）从过期dict中删除键；
	dictDelete（dict）从dict中删除键；
dbUnshareStringValue（db，key，o）：返回对象不共享的版本。
	如果refcount=1 且 编码为RAW字符串；
	创建一个o的副本；
	dbOverWrite存储key-o副本；
emptyDb（db）：清空数据库。
	清除每个db的dict和expires；
selectDB(client,id):切换client的数据库到id指向的数据库。
signalModifiedKey：键改动的钩子函数。
signalFlushedDb：刷新db的钩子函数。
flushdbCommand（client）：FLUSHDB。清空client指定的数据库。
	signalFlushedDb；
	dictEmpty清空dict；
	dictEmpty清空expires；
flushallCommand：FLUSHALL。清空所有数据库。
	signalFlushedDb；
	emptyDb清空数据库；
	取消RDB保存；
	更新RDB文件；
delCommand：DEL key [key ...]。批量删除键。
	对每个键：
		expireIfNeeded删除过期键；
		dbDelete删除键；
		signalModifiedKey;
existsCommand:EXISTS key。检测键是否存在。
	expireIfNeeded删除过期键；
	dbExists检查键是否存在；
selectCommand：SELECT index。选择下标0处的数据库。
	selectDb切换到index处的数据库；
randomkeyCommand：RANDOMKEY。随机返回一个key。
	dbRandomKey；
keysCommand：KEYS pattern。返回匹配pattern的key。
	遍历dict字典：
		将键名与pattern对比，匹配的输出；
scanCallback：scanfGenericCommand的回调函数。收集字典键和值。
parseScanCursorOrReply（client，o，cursor）：将o中的scan cursor存储在cursor中；
scanGenericCommand（client，o，cursor）：
	创建结果链表keys；
	o为NULL 或者o的类型为SET，HASH和ZSET；
	取COUNT选项参数；
	取MATCH选项参数；
	如果o的数据结构是哈希表：
		dictScan（cursor，scanCallback）取key，存储到keys，直到keys中有count个；
	如果o的数据结构是intset：
		intsetGet取出全部值，放入keys；
	如果o的数据结构是ziplist：
		ziplistGet和ziplistNext取出全部值，放入keys；
	对keys中的每个元素：
		过滤掉不满足pattern的元素；
		过滤掉超时的元素；
scanCommand：SCAN cursor [MATCH pattern] [COUNT count]。游标扫描。
	scanGenericCommand封装；
dbsizeCommand：DBSIZE。数据库key的个数。
	dictSize；
lastsaveCommand：LASTSAVE。最后一次同步磁盘的时间戳。
typeCommand：TYPE key。获取key的存储类型。
	返回key的类型字符串。
shutdownCommand：SHUTDOWN [NOSAVE] [SAVE]。关闭服务。
	prepareForShutdown；
renameGenericCommand：给键改名。
	源键名与目标键名不相同；
	lookupKeyWriteOrReply取出源键；
	lookupKeyWrite检查目标键，删除已有的键；
	将目标键-源值dictAdd加入数据库；
	dbDelete删除源键；
renameCommand：RENAME key newkey。给key重命名。
renamenxCommand：RENAMENX key newkey。新键不存在时，重命名。
	renameGenericCommandf封装；
moveCommand：MOVE key db。将当前数据库的 key 移动到给定的数据库 db 当中。
	selectDB切换到目标数据库dst；
	selectDB切换到源数据库src；
	如果src==dst，返回；
	lookupKeyWrite取出源对象；
	lookupKeyWrite取出目标对象，如果存在返回；
	dbAdd将键添加到dst；
	dbDelete从src中删除键；
removeExpires（db，key）：删除key的过期时间。
	dictFind在dict中查找key；
	dictDelete从expires中删除key；
setExpire（db，key，when）：设置key的过期时间。
	dictFind（dict）取出节点；
	dictReplaceRaw（expires）更新过期时间；
getExpire（db，key）：返回key的过期时间。
	dictFind（expires）取出节点；
	dictFind（dict）确认在dict中；
propagateExpire（db，key）：将键的过期时间，传播到AOF和附属节点。
	构造DEL命令；
	feedAppendOnlyFile传播到AOF；
	replicationFeedSlaves传播到附属节点；
expireIfNeeded（db，key）：如果key已经过期，将其从数据库中删除。
	getExpire取过期时间；
	propagateExpire传播过期时间；
	dbDelete删除key；
expireGenericCommand（client，basetime，unit）：
	lookupkeyRead取出键；
	setExpire设置超时时间；
	signalModifiedKey；
expireCommand:EXPIRE key seconds。设置超时时间。
expireatCommand:EXPIREAT key timestamp。设置超时时间戳。
pexpireCommand:PEXPIRE key milliseconds。设置超时时间毫秒。
pexpireatCommand:PEXPIREAT key milliseconds-timestamp。设置超时时间戳毫秒。
	expireGenericCommand封装；
ttlGenericCommand：返回键的剩余时间。
	lookupkeyRead取出键；
	getExpire取过期时间；
	计算剩余时间并返回；
ttlCommand：TTL key。获取剩余时间。
pttlCommand：PTTL key。获取剩余时间，毫秒。
	ttlGenericCommand封装；
persistCommand：PERSIST key。将key变成永久的。
	dictFind（dict）取出键；
	removeExpire删除过期键；
getKeysUsingCommandTable：返回命令参数的索引数组。
getKeysFromCommand：返回命令中key的位置数组；
	getKeysUsingCommandTable；
getKeysFreeResult：释放索引数组。
zunionInterGetKeys：从ZUNIONSTORE和ZINTERSTORE提取key。帮助函数。
evalGetKeys：从EVAL和EVALSHA提取key。帮助函数。
sortGetKeys：从SORT提取key。帮助函数。
slotToKeyAdd：将key添加到slot中，用于rehash集群。
slotToKeyDel：将key从slot中删除。
slotToKeyFlush：清空节点所有槽保存的所有键。
getKeysInSlot：返回slot的key。
delKeysInSlot：删除slot中的key。
countKeysInSlot：返回slot的key数量。

命令实现函数：

set函数：
	renameCommand：RENAME key newkey。给key重命名。
	renamenxCommand：RENAMENX key newkey。新键不存在时，重命名。
	moveCommand：MOVE key db。将当前数据库的 key 移动到给定的数据库 db 当中。
	
get函数：
	randomkeyCommand：RANDOMKEY。随机返回一个key。
	keysCommand：KEYS pattern。返回匹配pattern的key。
	scanCommand：SCAN cursor [MATCH pattern] [COUNT count]。游标扫描。
	typeCommand：TYPE key。获取key的存储类型。
超时函数：
	expireCommand:EXPIRE key seconds。设置超时时间。
	expireatCommand:EXPIREAT key timestamp。设置超时时间戳。
	pexpireCommand:PEXPIRE key milliseconds。设置超时时间毫秒。
	pexpireatCommand:PEXPIREAT key milliseconds-timestamp。设置超时时间戳毫秒。
	ttlCommand：TTL key。获取剩余时间。
	pttlCommand：PTTL key。获取剩余时间，毫秒。
	persistCommand：PERSIST key。将key变成永久的。
	
删除函数：
	flushdbCommand（client）：FLUSHDB。清空client指定的数据库。
	flushallCommand：FLUSHALL。清空所有数据库。
	delCommand：DEL key [key ...]。批量删除键。

检查函数：
	existsCommand:EXISTS key。检测键是否存在。

寻找函数：
	selectCommand：SELECT index。选择下标0处的数据库。

杂项函数：
	dbsizeCommand：DBSIZE。数据库key的个数。
	lastsaveCommand：LASTSAVE。最后一次同步磁盘的时间戳。
	shutdownCommand：SHUTDOWN [NOSAVE] [SAVE]。关闭服务。

通知功能：

  1. 键空间通知——某个键执行了什么命令；键事件通知——某个命令被什么键执行了。
  2. 要用到pubsub模块的

通知相关的函数：

keyspaceEventsStringToFlags：对传入的字符串参数进行分析， 给出相应的 flags 值。
keyspaceEventsFlagsToString：将flags转换成参数字符串。
notifyKeyspaceEvent：键空间事件通知函数。
	pubsubPublishMessage发送键空间通知；
pubsubPublishMessage发送键事件通知；
