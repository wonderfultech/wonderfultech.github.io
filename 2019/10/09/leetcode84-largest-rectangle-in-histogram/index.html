leetcode84 柱状图中最大的矩形 Largest Rectangle in Histogram

Posted on 2019年10月9日2019年12月4日by jasonpeng

解法一 单调队列法

方法不是很直观。

从左向右遍历。

位置0单独构成矩形。

位置1处向左延申到位置-1。

位置2不能向左延申，是否能向右延申不知道。

位置3不能向左延申，是否能向右延申不知道。

位置4能向左延申到位置1， 是否能向右延申不知道。

位置5不能向左延申，是否能向右延申不知道。

位置6能向左延申到位置1， 是否能向右延申不知道。

...

从上面可以看出，每个位置i能向左延申的位置j，heights[j] < heights[i]。

可以确定的是位置j+1到位置i，一定能构成高度为heights[i]的矩形。位置j+1就是矩形的左边界。

可以用单调队列Q记录这样的位置j，这些位置j都是某一个矩形的左边界。队列中位置的高度依次从左向右依次递增。

例如：在上图中，某个时刻Q中的值为[1,2,3]。

可以看出位置1是位置2能构成矩形的左边界。

位置2是位置3能构成矩形的左边界。

什么时候确定一个矩形的右边界了？

在上例中，Q的尾部为位置3，遍历到位置4，heights[4] < heights[3]， heights[4] < heights[2]， heights[4] > heights[1] 。

这说明，位置2和位置3不能与位置4构成以heights[2]和heights[3]为高度的矩形。

这时要计算这些矩形的面积。面积 = 高度 x 宽度。

高度 = Q中位置的高度。例如，heights[3]和heights[2]。

宽度 = 新位置 与 Q尾部前一个位置的距离 - 1。例如，高度为heights[3]的矩形的宽度 = 位置4 - 位置2 - 1 . 高度为heights[2]的矩形的宽度 = 位置4 - 位置1 - 1 .

整体思路：

Q：队列;
heights:高度数组;
for(i < heights.length){
    while(heights[Q.tail] >= heights[i]){
         h = Q.tail;
         Q尾部出队列;
         if(Q为空){
             w = i;
         }else{
             w = i - Q.tail - 1;
         }
         取w * h的最大值；
   }
}

完整代码：

    int largestRectangleArea(vector<int>& heights) {
        heights.push_back(0);
        vector<int> Q;
        int ans = 0;
        for(int i=0;i<heights.size();++i){
            while(Q.size() && heights[i] <= heights[Q.back()]){
                int h = heights[Q.back()];
                Q.pop_back();
                int w;
                if(Q.size()){
                    w = i - Q.back() - 1;
                }else{
                    w = i;
                }
                ans = max(ans,w * h);
            }
            Q.push_back(i);
        }
        return ans;
    }

解法二 左右边界法

直观的方法。

求出以位置4的高度为高度的矩形。

容易看出，左边能到位置2，位置1的heights[1] < heights[4]。

右边能到位置5，因为heights[6] < heights[4]。

这样，对每个位置i，找到其左边最近的高度小于heights[i]和右边最近的高度小于heights[i]的位置，称之为左边界和右边界。算面积就很容易了。

以找出左边界的为例。

例如，位置3的左边界是位置2，位置2的左边界是位置1，位置1的左边界是位置-1，位置0的左边界是-1.

那么，位置4的左边界怎么算？

因为heights[3] > heights[4]，那么位置3的左边界，可能是位置4的左边界。

位置3的左边界是位置2，heights[2] > heights[4]，那么位置2的左边界，可能是位置4的左边界。

位置2的左边界是位置1，heights[1] > heights[4] ，位置4的左边界肯定是位置1.

由于存在这样的递推关系，可以简化求左边界的计算量。

整体思路：

left[i]:位置i的左边界；
right[i]:位置i的右边界；
left[0]=-1;
for(i in [1,...,N-1]){
    left[i] = i-1;
    while(left[i] >=0  && heights[left[i]] >= heights[i]){
        left[i] = left[left[i]];
    }
}
right[N-1] = N;
for(i in [N-2,...,0]){
    right[i] = i+1;
    while(right[i] >=0  && heights[right[i]] >= heights[i]){
        right[i] = right[right[i]];
    }
}
for(i in [0,...,N-1]){
    取最大值(right[i] - left[i] - 1) * heights[i];
}

完整代码：

    int largestRectangleArea(vector<int>& heights) {
        const int N = heights.size();
        int ans = 0;
        if(N == 0) return 0;
        vector<int> left(N),right(N);
        left[0] = -1;
        for(int i=1;i<N;++i){
            left[i] = i - 1;
            while(left[i] >=0 && heights[left[i]] >= heights[i]){
                left[i] = left[left[i]];
            }
        }
        right[N-1] = N;
        for(int i=N-2;i>=0;--i){
            right[i] = i+1;
            while(right[i] < N && heights[right[i]] >= heights[i]){
                right[i] = right[right[i]];
            }
        }
        for(int i=0;i<N;++i){
            ans =  max(ans,(right[i] - left[i] -1)*heights[i]);
        }
        return ans;
    }
