leetcode23 合并K个排序链表 Merge k Sorted Lists

Posted on 2019年10月6日2019年12月4日by jasonpeng

Table of Contents

  • 解法二 折半归并-递归法

解法一 折半归并-迭代法

可以将问题简化为两两归并。每归并一次，链表就少一个，最终必然结束。

但是，可以N个链表，每次分成两组，组之间一一对应归并，这样一次就可少一半的链表。

例如：链表索引:[0,1,2]

分成两组 [0，1]和[2]，0和2对应的链表归并到0。1没有对应的链表。

归并后链表为[0,1]，再需1次归并即可。

例如：链表索引：[0，1，2，3，4]

第一轮，分成两组[0,1,2]和[3,4]。0和3归并到0，1和4归并1，2不归并。归并后链表为[0,1,2]。

接着按上面的例子归并。

例如：链表索引：[0，1，2，3]

第一轮，分成两组[0,1]和[2,3]。0和2归并到0，1和3归并到1。归并后链表为[0,1]。

从上面的例子中，可发现当链表N为奇数时，第一组比第二组多一个，便于后续处理。

数字化后，当N为奇数，第一组个数为a，那么第二组个数为a-1。有a+a-1=N，则有a=(N+1)/2。

第一组为:[0,1,2,...,a-1]，第二组为[a,a+1,a+2,...,min(2a-1,len-1]。

第一组下标+a，就对应到第二组中的链表。

归并后链表个数变成N=a个。

完整思路：

N:链表个数;
A:含有N个链表的数组;
while(N > 1){
    a = (N+1)/2;
    for(i in [0,1,2,...,a-1]; i+a in [a,a+1,a+2,...,min(2a-1,len-1)]){
        归并链表:i 和 i+a 到链表i;
    }
    N = a;
}
A[0]是归并结果;

剩下如何归并两个有序链表。

这是有固定套路的。

设A和B分别是链表的头部。

归并关键步骤：

if(B为空节点 或  A不为空节点 且 A->val <= B->val){
    取A->val;
    A向前一步;
}else{
    取B->val;
    B向前一步;
}

归并代码：

    ListNode* merge(ListNode* A,ListNode* B){
        ListNode head(-1),*p=&head;
        while(A || B){
            if(!B || A && A->val <= B->val){
                p->next = A;
                A = A->next;
            }else{
                p->next = B;
                B = B->next;
            }
            p = p->next;
        }
        return head.next;
    }

完整代码：

    ListNode* mergeKLists(vector<ListNode*>& lists) {
        int len = lists.size();
        if(len < 1) return NULL;
        while(len>1){
            int m = (len + 1) / 2;
            for(int i=0;i<m && i+m < len;++i){
                lists[i] = merge(lists[i],lists[i+m]);
            }
            len = m;
        }
        return lists[0];
    }

解法二 折半归并-递归法

解法一，这种折半成两组归并的方法，有点类似于归并排序。不过，在这里，每个索引代表的是一个链表，而不是一个节点。

归并排序的思路，是将链表分成前后两段，每段内部在递归合并。最终合并两个归并好的有序链表。

解法二比解法一要简洁。

思路:

A:含有N个链表的数组;
递归归并A[x,y)的区间内的链表;
dfs(A,x,y){
    if(y ==  x) 返回空链表;
    if(y ==  x+1) 返回A[x];
    if(y == x+2) 返回 merge(A[x],A[x+1]);
    m = x + (y - x) /2 ;
    返回:merge(dfs(A,x,m),dfs(A,m,y));
}

完整代码：

    ListNode* mergeKLists(vector<ListNode*>& lists) {
        return dfs(lists,0,lists.size());
    }
    
    //[x,y)
    ListNode* dfs(vector<ListNode*>& lists,int x, int y){
        if(x == y) return NULL;
        if(x+1 == y) return lists[x];
        if(x+2 == y) return merge(lists[x],lists[x+1]);
        int m = x+(y-x)/2;
        return merge(dfs(lists,x,m),dfs(lists,m,y));
    }
