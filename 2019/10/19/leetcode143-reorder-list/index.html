leetcode143 重排链表 Reorder List

Posted on 2019年10月19日2019年12月4日by jasonpeng

leetcode 数据库题目全部题解

解法一 拆分法

思路：先将链表拆分成前后两半A和B。后一半B逆转，成C。再将A和C交叉合并。

例如：1->2->3->4->5

拆分成A=1->2->3 ，B=4->5。

B逆转成C=5->4。

A和C合并成D=1->5->2->4->3。

第一步，用快慢指针拆分链表为两半。

例如：1->2->3->4->5

slow -> 1 fast->1

slow每次走一步。fast每次走两步。

直到fast为空。

第一次走，slow->2 fast->3

第二次走，slow->3 fast->5

第三次走，slow->4 fast->NULL

此时slow->4 就是后半链表的起点.

思路：

slow = head, fast = head;
while(fast不空){
    slow = slow.next;
    fast = fast.next;
    if(fast不空) fast = fast.next;
}
slow指向后半节点;

第二步，后一半链表逆转。

将第一步产生的链表，节点逆序。

例如：slow->4->5

逆序后slow->5->4。

用递归方法逆转链表T。

先递归逆转节点T后部的链表。然后将A节点挂到已逆转部分的尾部。

思路：

reverse_list(A){
    if(A为空 或 A为单节点) 返回A;
    rest  = reverse_list(A的后部链表);
    if(rest不空) A.next.next = A;
    返回rest;
}

代码：

    ListNode* reverse_list(ListNode* A){
        if(!A || !A->next) return A;
        ListNode* r = reverse_list(A->next);
        if(A->next) A->next->next = A;
        A->next = NULL;
        return r;
    }

第三步，两部分交叉合并。

交叉合并如此进行。

例如：A=1->2->3 C=5->4。

第一次，从A取1. ANS=1

第二次，从C取5. ANS=1->5

第三次，从A取2. ANS=1->5->2

第四次，从A取4. ANS=1->5->2->4

第五次，从A取3. ANS=1->5->2->4->3

思路：

first前半链表;
second后半链表;
ANS结果链表;
c = 1
while(first 且 second){
    if(c){
        ANS += first;
        first = first.next;    
    }else{
        ANS += second;
        second = second.next;
    }
    if(c) c=0;
    else c=1;
}
//剩余的节点
if(first){
    ANS += first;
}
if(second){
    ANS += second;    
}

综上完整代码：

    void reorderList(ListNode* head) {
        if(!head || !head->next || !head->next->next) return ;
        ListNode* slow = head,* fast = head,* pre_of_slow = NULL;
        while(fast){
            pre_of_slow = slow;
            slow = slow->next;
            fast = fast->next;
            if(fast) fast = fast->next;
        }
        //split the rest;
        pre_of_slow->next = NULL;
        //reverse the rest;
        ListNode * r = reverse_list(slow);
        //print(head);
        //print(r);
        //merge them again
        ListNode* ans = NULL,* T = NULL;
        ListNode * f = head;
        int c = 1;
        while(r && f){
            if(c){
                if(!ans){
                    ans = f;
                    T = ans;
                }else{
                    T->next = f;
                    T = T->next;
                }
                f = f->next;
            }else{
                T->next = r;
                r = r->next;
                T = T->next;
            }
            
            c = c ^ 1;
        }
        if(f){
            T->next = f;
            T = T->next;
        }
        if(r){
            T->next = r;
            T = T->next;
        }
        T->next = NULL;
    }
};
