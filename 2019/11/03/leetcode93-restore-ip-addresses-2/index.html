leetcode93 复原IP地址 Restore IP Addresses

Posted on 2019年11月3日2019年12月4日by jasonpeng

leetcode 数据库题目全部题解

解法一 回溯法

ip地址的特点: 必须是三个点, 分割出四段. 每段都在[0,255]之间.

字符串s转成ip地址的要求:

  1. s中的字符在ip串中不能少.
  2. ip中必须有三个点, 四个段.
  3. 每个段最少一个字符, 最多三个字符, 且段值在[0,255]
  4. 字符'0'只能单独成段.

例如:"25525511135"

ip "255.255.11.13" 少了最后的5

ip "255.255.11.13.5" 四个点

ip "255.255.1.1135" 1135超过255

例如:"01234"

ip "0.1.234" 只有两个点

思路: 通过三点进行回溯减枝. 根据段中字符个数和段值范围对段进行枚举.

path[cur];存储临时ip串;
dot=0;点的个数
dfs(s,cur,path,dot){
    if(dot > 3) 减枝;
    if(s空串){
        //s全部提取完了
        if(dot == 3) 正确的ip串;
    }else{
        //第二段开始, 段前有'.'
        if(cur) path[cur]='.',++cur,++dot;
        //最多三个字符组成一个段, 如果字符是'0', 只能单独成段
        cnt = min(3,s.length);
        if(s[0] == '0') cnt=1;
        //成段
        for(len in [1,cnt]){
            sub = s[0,len-1];
            if(sub in [0,255]){
                path[cur,cur+len-1]=sub;
                dfs(s[len,...,N],cur+len,path,dot);
            }
        }
    }
}

代码:

    vector<string> restoreIpAddresses(string s) {
        vector<string> ans;
        vector<char> path(s.size()+10);
        dfs(s,0,path,0,ans);
        return ans;
    }
    
    void dfs(const string& s,
             int cur,
             vector<char>& path,
             int dot,
             vector<string>& ans){
        if(dot > 3) return ;
        if(s.empty()){
            if(dot == 3){
                ans.push_back(string(path.begin(),path.begin()+cur));
            } 
        }else{
            if(cur) path[cur++] = '.',++dot;
            
            int c=3;
            if(s[0] == '0') c=1;
            c = min((int)s.size(),c);
            for(int i=0;i<c;++i){
                string sub=s.substr(0,i+1);
                int d=stoi(sub);
                if(d>=0 && d<=255){
                    for(int j=0;j<i+1;++j) path[cur+j]=s[j];
                    dfs(s.substr(i+1),cur+i+1,path,dot,ans);
                }
            }
        }
    }
