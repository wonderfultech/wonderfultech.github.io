leetcode198 打家劫舍 House Robber

解法一 动态规划

在房屋i处有两种选择, 偷和不偷.

设d[i][0]表示遍历到房屋i处, 不偷时的最大值.

设d[i][1]表示遍历到房屋i处, 偷时的最大值.

最终结果是max(d[i][0],d[i][1]).

当 在i处不偷 时, 此处的最大值,只能是i-1处的最大值.

即 d[i][0]=max(d[i-1][0],d[i-1][1])

当 在i处偷 时, 在i-1处不能偷, 遂考虑i-2处的最大值.

即d[i][1]=A[i] + max(d[i-1][0] ,max( d[i-2][0],d[i-2][1]) )

d[i][2]=0;
for(i < N){
    if(i==0){
        d[i][0]=0;
        d[i][1]=A[0];
    }else{
        d[i][0]=max(d[i-1][0],d[i-1][1]);
        d[i][1]=A[i] + max(d[i-1][0] ,max( d[i-2][0],d[i-2][1]) )
    }
}
max(d[N-1][0],d[N-1][1])是结果;

代码:

    int rob(vector<int>& nums) {
        if(nums.empty()) return 0;
        vector<vector<int>> d(nums.size(),vector<int>(2,0));
        for(int i=0;i<nums.size();++i){
            if(!i){
                d[i][0]=0;
                d[i][1]=nums[i];
            }else{
                d[i][0]=max(d[i-1][0],d[i-1][1]);
                int i_minus_two = (i >= 2) ? max(d[i-2][0],d[i-2][1]) : 0;
                d[i][1]=nums[i] + max(d[i-1][0], i_minus_two);
            }
        }
        return max(d.back()[0],d.back()[1]);
    }

化简

上面的解法中.

当 在i处偷 时, 在i-1处不能偷, 遂考虑i-2处的最大值.

即d[i][1]=A[i] + max(d[i-1][0] ,max( d[i-2][0],d[i-2][1]) )

实际上在i-2的情况包含在i-1的情况中了.在i-1处一定不能偷.

即 d[i][1]=A[i] + d[i-1][0]

d[i][2]=0;
for(i < N){
    if(i==0){
        d[i][0]=0;
        d[i][1]=A[0];
    }else{
        d[i][0]=max(d[i-1][0],d[i-1][1]);
        d[i][1]=A[i] + d[i-1][0];
    }
}
max(d[N-1][0],d[N-1][1])是结果;

代码:

    int rob(vector<int>& nums) {
        if(nums.empty()) return 0;
        vector<vector<int>> d(nums.size(),vector<int>(2,0));
        for(int i=0;i<nums.size();++i){
            if(!i){
                d[i][0]=0;
                d[i][1]=nums[i];
            }else{
                d[i][0]=max(d[i-1][0],d[i-1][1]);
                d[i][1]=nums[i] + d[i-1][0];
            }
        }
        return max(d.back()[0],d.back()[1]);
    }

去掉递推数组

又由于d[i]只与d[i-1]有关. 这样可用两个变量a,b表示在位置i偷与不偷的情况.

a=0;不偷
b=0;偷
for(i < N){
    if(i==0){
        a=0;
        b=A[0];
    }else{
        t=max(a,b);
        b=A[i] + a;
        a=t;
    }
}
max(a,b)是结果;

代码:

    int rob(vector<int>& nums) {
        if(nums.empty()) return 0;
        int a,b;
        for(int i=0;i<nums.size();++i){
            if(!i){
                a=0;
                b=nums[i];
            }else{
                int t=max(a,b);
                b=nums[i] + a;
                a=t;
            }
        }
        return max(a,b);
    }
