leetcode101 对称二叉树 Symmetric Tree

Posted on 2019年11月30日2019年12月4日by jasonpeng

解法一 递归法1 直观的思路, 1,先遍历整个二叉树,将每层的节点存起来. 2,对每层节点检查是否对称. 例如: 1 / \ 2 2 \ \ 3 3 第一层[1] 第二层[2,2] 第三层[null,3,null,3] 第三层不对称,因此不是对称二叉树. 代码: 解法二 递归法2 能否同时遍历两棵树? 如果直接从根节点开始,是无法同时遍历的. 但是如果从根节点的两个子节点开始遍历,同时遍历左右子树是可行的. 在遍历的同时,进行比较. 例如: 1 / \ 2 2 \ \ 3 3 从根节点1的两个子树(2,2)开始. 由于要考察对称下,递归遍历考虑两种情况. 递归遍历,第一个2的左子树和第二个2的右子树. 递归遍历,第一个2的右子树和第二个2的左子树. 代码: 解法三 广度优先搜索 考虑将解法二改成迭代法. 基于层次遍历. 从根节点的左右子树开始, 将要比较的左右子树. 存入队列中. 不断从队列中取节点, 比较匹配情况, 然后再拓展. 代码: Hits: 1

leetcode448 找到所有数组中消失的数字 Find All Numbers Disappeared in an Array

Posted on 2019年11月30日2019年12月4日by jasonpeng

本题与 442. 数组中重复的数据 的解法相同. 解法一 交换法 关键是用好条件： 1 ≤ a[i] ≤ n （n为数组长度） 。 a[1]-1在[0,n-1]中。将a[i]放在位置a[i]-1是可行的。 遍历元素时， 当 a[i] != i+1的，就将其交换到正确的位置a[i]-1。 此时，当 a[i] == a[a[i]-1]时，表示a[i]有重复。 遍历完后，再遍历一次。所有 a[i] != i+1 的i+1就是结果。 代码: Hits: 7

Posted on 2019年11月30日2019年12月4日by jasonpeng

解法一 动态规划 在房屋i处有两种选择, 偷和不偷. 设d[i][0]表示遍历到房屋i处, 不偷时的最大值. 设d[i][1]表示遍历到房屋i处, 偷时的最大值. 最终结果是max(d[i][0],d[i][1]). 当 在i处不偷 时, 此处的最大值,只能是i-1处的最大值. 即 d[i][0]=max(d[i-1][0],d[i-1][1]) 当 在i处偷 时, 在i-1处不能偷, 遂考虑i-2处的最大值. 即d[i][1]=A[i] + max(d[i-1][0] ,max( d[i-2][0],d[i-2][1]) ) 代码: 化简 上面的解法中. 当 在i处偷 时, 在i-1处不能偷, 遂考虑i-2处的最大值. 即d[i][1]=A[i] + max(d[i-1][0] ,max( d[i-2][0],d[i-2][1]) ) 实际上在i-2的情况包含在i-1的情况中了.在i-1处一定不能偷. 即 d[i][1]=A[i] + d[i-1][0] 代码: 去掉递推数组 又由于d[i]只与d[i-1]有关. 这样可用两个变量a,b表示在位置i偷与不偷的情况. 代码: Hits: 1
