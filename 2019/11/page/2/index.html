leetcode415 字符串相加 Add Strings

Posted on 2019年11月29日2019年12月4日by jasonpeng

解法一 按位加法 首先明确数字串是高位数字放在底索引处. “123”. 1是高位数字. 整数相加, 因为有进位, 从数字的低位数字相加, 再向高位数字进位. 也就是索引要从高到低遍历串. 例如: “123” + “7” 先3+7=10,进位1,值为0. 再2+1=3, 进位0,值为3 再1+0=1,进位0,值为1. 结果是130. 代码: Hits: 3
