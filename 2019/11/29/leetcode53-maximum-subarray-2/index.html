leetcode53 最大子序和 Maximum Subarray

解法一 前缀和法

定义前缀和S[i]=A[0]+A[1]+...+A[i].

S[0]=A[0].

通过枚举区间[i,j], 求最大的连续子数组和 .

[i,j]中元素的和 = S[j]-S[i-1]. 然后求max( [i,j]中元素的和 ).

max( [i,j]中元素的和 ) = max(S[j]-S[i-1]).

当固定j, 而枚举i时, max(S[j]-S[i-1]) =S[j] - min(S[i-1]).

遍历前缀和S[j]时,只要记录其左边的最小前缀和min(S[i-1]), 就能算出区间[i,j]的最大值.

另外,前缀和可以遍历的同时计算.

这让线性处理本题成为可能.

S=A[0];前缀和S[j]
min_S=min(0,S);最小前缀和
ans=S;结果
for(1 < i < N){
    S += A[i];算前缀和
    ans=max(ans,S-min_S);
    min_S=min(min_S,S);算最小前缀和
}

代码:

    int maxSubArray(vector<int>& nums) {
        const int N = nums.size();
        int s=nums[0];
        int min_s=min(s,0);
        int ans = s;
        for(int i=1;i<N;++i){
            s += nums[i];
            ans = max(ans,s - min_s);
            min_s = min(min_s,s);
        }
        return ans;
    }

解法二 动态规划

直接考虑区间[i,j]内的 最大子数组和 .

一个障碍是 最大子数组和 对应的区间不一定就以A[i]或A[j]为结尾. 区间[i,j]内的结果无法向两边拓展.

限制固定i为0. 并且只考虑以A[j]为结尾的最大子数组和.

因为 最大子数组和 对应的区间 总要以某个元素为结尾, 将这个结尾的元素指定为A[j]. 这样的指定不失一般性.

设d[j]表示以元素A[j]为结尾的最大子数组和.

已知d[j-1], 要考虑 以元素A[j]为结尾的最大子数组 能否扩展到A[j-1]及其之前的位置. 要判断d[j-1]+A[j] 与 A[j]的大小.

当 d[j-1]+A[j] > A[j] 时, 说明 以元素A[j]为结尾的最大子数组 能扩展到A[j-1]及其之前的位置.

否则, 显然没必要.

d[i]=0;
d[0]=A[0];
ans=A[0];结果
for(1 < i < N){
    d[i] = max(d[i-1]+A[i],A[i]);
    ans=max(ans,d[i]);
}

代码:

    int maxSubArray(vector<int>& nums) {
        const int N = nums.size();
        vector<int> d(N,0);
        d[0] = nums[0];
        int ans = d[0];
        for(int i=1;i<N;++i){
            d[i] = max(d[i-1]+nums[i],nums[i]);
            ans = max(ans,d[i]);
        }
        return ans;
    }

去掉一维数组

可以看出d[j]只与d[j-1]相关. 其实递推只需要一个变量即可.

d=A[0];
ans=A[0];结果
for(1 < i < N){
    d = max(d+A[i],A[i]);
    ans=max(ans,d);
}

代码:

    int maxSubArray(vector<int>& nums) {
        const int N = nums.size();
        int d = nums[0];
        int ans = d;
        for(int i=1;i<N;++i){
            d = max(d+nums[i],nums[i]);
            ans = max(ans,d);
        }
        return ans;
    }

解法三 分治法

直接考虑区间[i,j]内的 最大子数组和 .

解法二是固定一端,并枚举另一端.

换个思路, 用粗暴的方法, 将区间[i,j]分成两半. [i,m]和[m+1,j].

那么, 最大子数组和,出现三种可能.

1, 全在[i,m]中;

2, 全在[m+1,j]中;

3, 横跨中线m;

这里理解上的难点是第1和2种情况, 不一定能覆盖中间位置A[m]和A[m+1].

但是第3种情况必然以 A[m]和A[m+1] 为边界. 这是重要区别.

这样,分成两半后, 对两个小区间,就可以递归求解了.

[i,j]的值 = 三种情况的间的最大值.

//区间[x,y)
dfs(A,x,y){
    if(x+1 == y) 单元素A[x],返回;
    m=(x+y)/2;
    l=dfs(A,x,m);
    r=dfs(A,m,y);
    rmax = 求从A[m]向右扩展的最大子序列和;
    lmax = 求从A[m-1]向左扩展的最大子序列和;
    返回max(lmax+rmax,max(l,r));
}

代码:

    int maxSubArray(vector<int>& nums) {
        return dfs(nums,0,nums.size());
    }
    
    //[x,y)
    int dfs(const vector<int>& A,int x,int y){
        if(x+1 == y) return A[x];
        int m = x + (y - x)/2;
        int l=dfs(A,x,m);
        int r=dfs(A,m,y);
        int i=m;
        int rr=0,rmax=A[i];
        while(i < y) rmax=max(rmax, rr += A[i++]);
        i=m-1;
        int ll=0,lmax=A[i];
        while(i >= x) lmax=max(lmax,ll += A[i--]);
        return max(lmax+rmax,max(l,r));
    }
