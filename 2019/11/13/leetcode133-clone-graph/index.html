leetcode133 克隆图 Clone Graph

Posted on 2019年11月13日2019年12月4日by jasonpeng

leetcode 数据库题目全部题解

解法一 深度优先搜索

无向简单图的复制。

图中可能有环，增加了复杂性。

对每个节点A，复制其副本A'。

构造hash<A,A'>。构造vis[A]=1去除访问重复。

然后递归复制A的邻接点{A1,...,AN}。

当遍历一个节点AX，AX已经访问过时，表示图中已经有环的存在。AX的复制已经完成了。

从思路中可看出，hash<A,A'>和vis[A]=1，是先于递归复制的。

例如：

复制1，hash[1]=1',vis[1]=1;

递归复制4，hash[4]=4',vis[4]=1;

递归复制3，hash[3]=3',vis[3]=1;

递归复制2，hash[2]=2',vis[2]=1;

递归复制1，但是vis[1]=1，那么返回hash[1]=1'，即可。

完整思路：

dfs(A,hash,vis){
    if(A为空) 返回空;
    if(vis[A] == 1) 返回hash[A];
    节点A的副本A';
    hash[A]=A';
    vis[A]=1;
    for(邻居节点i in A的所有邻居){
        A'的邻居节点i = dfs(邻居节点i,hash,vis);
    }
    返回A';
}

代码：

    Node* cloneGraph(Node* node) {
        unordered_map<Node*,int> vis;
        unordered_map<Node*,Node*> hash;
        
        return dfs(node,vis,hash);
    }
    
    Node* dfs(Node* node,
              unordered_map<Node*,int>& vis,
              unordered_map<Node*,Node*>& hash){
        if(!node) return node;
        if(vis.count(node)) return hash[node];
        Node* nd=new Node(node->val,vector<Node*>(node->neighbors.size(),NULL));
        hash[node] = nd;
        vis[node] = 1;  
        for(int i=0;i<node->neighbors.size();++i){
            Node* nb = node->neighbors[i];
            Node* new_nb=dfs(nb,vis,hash);
            nd->neighbors[i] = new_nb;
        }
        return nd;
    }
