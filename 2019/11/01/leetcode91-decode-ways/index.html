leetcode91 解码方法 Decode Ways

Posted on 2019年11月1日2019年12月4日by jasonpeng

解法一 回溯法

思路: 在位置i, 子串A[i,...,N-1]的解码数, 两种情况.

当数字A[i]在区间[1,26]中时, 说明A[i]单独解码.

那么,递归计算串A[i+1,...,N-1]的方案数.

当数字A[i,i+1]在区间[1,26]中. 说明A[i,i+1]单独解码.

那么,递归计算串A[i+2,...,N-1]的方案数.

总之,位置i的方案数 = 位置i+1的方案数 + 位置i+2的方案数

例如: "226"

A[2]=6, 单独解码. 方案数1.

A[1]=2,单独解码 . 方案数 = A[2]的方案数=1.

A[1,2]=26,单独解码 . 方案数 = A[3]的方案数=1.

A[1]的总方案数 = 2.

A[0]=2, 单独解码. 方案数 = A[1]的方案数1=2

A[0,1]=22, 单独解码. 方案数 = A[2]的方案数1=1

A[0]的总方案数 = 3.

完整思路:

hash<串,方案数>; 记忆化存储
dfs(s,hash){
    if(s in hash) 返回hash[s];
    if(s空串){
        hash[s] = 1;
        返回1;
    }else for(len in [1,2]){
        sub = s[i,i+len-1];
        if(sub in [1,26]){
            hash[s] += dfs(s[i+len,...,N-1],hash);
        }
    }
    返回hash[s];
}

代码:

    int numDecodings(string s) {
        if(s.empty()) return cnt;
        unordered_map<string,int> hash;
        return dfs(s,hash);
    }
    
    int dfs(const string& s,unordered_map<string,int>& hash){
        if(hash.count(s)) return hash[s];
        int& ans = hash[s];
        if(s.empty()){
            return ans = 1;
        }else for(int len=1;len<=2;++len){
            if(len > 1 && s[0]=='0') continue;
            string sub = s.substr(0,len);
            int x = stoi(sub);
            if(1 <= x && x <= 26){
                if(len <= s.size()) {
                    string next=s.substr(len);
                    int r = dfs(next,hash);
                    ans += r;
                }
            }
        }
        return ans;
    }

解法二 动态规划

将解法一转成动态规划解法.

设d[i]表示子串s[i,...,N-1]的解码方案数.

按照解法一, 分成两种情况.

当数字A[i]在区间[1,26]中时, 说明A[i]单独解码.

那么, d[i] += d[i+1]

当数字A[i,i+1]在区间[1,26]中. 说明A[i,i+1]单独解码.

那么, d[i] += d[i+2]

完整思路:

d[i]=0;
d[s.length] = 1;
for(i in [s.length-1,...,0]){
    for(len in [1,2]){
        sub = s[i,i+len-1];
        if(sub in [1,26]){
            d[i] += d[i+len];
        }
    }
}
结果是d[0]

代码:

    int numDecodings(string s) {
        if(s.empty()) return 0;
        vector<int> d(s.size()+1,0);
        d[s.size()] = 1;
        for(int i=s.size()-1;i>=0;--i){
            int& ans=d[i];
            ans = 0;
            for(int len=1;len<=2;++len){
                if(len > 1 && s[i]=='0') continue;
                string sub = s.substr(i,len);
                int x = stoi(sub);
                if(1 <= x && x <= 26){
                    if(i+len <= s.size()) {
                        ans += d[i+len];
                    }
                }
            }
        }
        return d[0];
    }

常数空间

容易发现,在位置i,只需向前看到位置i+1,i+2. 那么, 存储空间是最多需要3个位置. 其它位置可以复用.

思路:

d[0,...,2]=0;
d[s.length  % 3] = 1;
for(i in [s.length-1,...,0]){
    for(len in [1,2]){
        sub = s[i,i+len-1];
        if(sub in [1,26]){
            d[i%3] += d[(i+len)%3];
        }
    }
}
结果是d[0]

代码:

    int numDecodings(string s) {
        if(s.empty()) return 0;
        vector<int> d(3,0);
        d[s.size() % 3] = 1;
        for(int i=s.size()-1;i>=0;--i){
            int& ans=d[i % 3];
            ans = 0;
            for(int len=1;len<=2;++len){
                if(len > 1 && s[i]=='0') continue;
                string sub = s.substr(i,len);
                int x = stoi(sub);
                if(1 <= x && x <= 26){
                    if(i+len <= s.size()) {
                        ans += d[(i+len) % 3];
                    }
                }
            }
        }
        return d[0];
    }
